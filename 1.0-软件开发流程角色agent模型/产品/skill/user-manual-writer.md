# Skill: user-manual-writer

## 基本信息
- **名称**: user-manual-writer
- **版本**: 1.0.0
- **所属部门**: 产品部
- **优先级**: P0

## 功能描述
根据PRD和UserStory文档，自动生成面向最终用户的操作手册。包含系统功能概述、各模块操作指南、常见问题FAQ等内容，帮助用户快速上手使用系统。

## 触发条件
- 命令触发: `/user-manual-writer`
- 自然语言触发:
  - "生成用户手册"
  - "编写使用文档"
  - "产出操作指南"
  - "生成帮助文档"

## 输入参数
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| prd_path | string | 是 | PRD文档路径 |
| userstory_path | string | 否 | UserStory文档路径 |
| screenshots_path | string | 否 | 系统截图/原型图目录 |
| target_users | string | 否 | 目标用户群体描述 |
| language | string | 否 | 输出语言，默认中文 |

## 执行流程
1. **文档解析** - 读取PRD和UserStory文档，提取功能模块和操作流程
2. **用户角色分析** - 识别不同用户角色及其操作场景
3. **功能梳理** - 按模块整理功能清单和操作步骤
4. **文档生成** - 生成用户手册、快速入门指南、FAQ文档
5. **校验完善** - 检查文档完整性，补充缺失内容

## 二、快速入门
### 2.1 登录系统
1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 选择当前班次和工作站（如需要）

### 2.2 首页导航
登录后进入首页，显示：
- 待办任务列表
- 今日生产进度
- 消息通知

### 2.3 常用操作
#### 扫码开始作业
1. 进入"装配作业"模块
2. 扫描工单二维码
3. 确认工序信息
4. 点击"开始作业"

#### 提交质量检验
1. 进入"质量管理"模块
2. 扫描产品条码
3. 填写检验结果
4. 提交检验记录

## 三、功能模块操作指南

### 3.1 生产工单管理
#### 3.1.1 功能概述
生产工单管理模块用于创建、下发、跟踪生产工单，支持工单的完整生命周期管理。

#### 3.1.2 创建工单
1. 点击"新建工单"按钮
2. 选择产品和产线
3. 输入计划数量和计划日期
4. 选择工艺路线
5. 点击"保存并下发"

#### 3.1.3 注意事项
- 工单下发后不可修改产品信息
- 计划数量不能超过产线日产能上限
- 需先配置工艺路线才能创建工单

## 四、常见问题FAQ

### Q1: 扫码提示"条码不存在"怎么办？
**A**: 请检查条码格式是否正确，确认条码已在系统中登记。如问题仍存在，请联系系统管理员。

### Q2: 如何修改已完成的作业记录？
**A**: 已完成的作业记录不可直接修改。如需更正，请提交变更申请，经审批后由管理员处理。

## 质量标准
- 功能覆盖率 ≥ 95%（所有PRD定义的功能都有操作说明）

## 依赖工具
- Read - 读取PRD和UserStory文档
- Write - 输出用户手册文档
- Grep - 搜索功能定义和用户故事
- AskUserQuestion - 澄清文档内容

## 注意事项
- 用户手册语言应简洁易懂，避免技术术语
- 操作步骤应具体可执行，包含按钮位置、输入格式等
- FAQ应从用户视角提问，而非技术视角
- 如有系统截图，应与文字说明配合使用
- 需要定期更新以保持与系统版本同步
