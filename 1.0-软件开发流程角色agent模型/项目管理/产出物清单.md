## 产出物详细说明

### 1. 项目排期表（Excel）

**包含Sheet**:
| Sheet名称 | 内容 |
|----------|------|
| 任务清单 | 任务ID、名称、日期、工期、负责人、优先级、前置任务、状态 |
| 工时汇总 | 人员负载率、总工时、可用工时 |
| 里程碑 | 里程碑名称、日期、验收标准、状态 |
| 风险清单 | 风险项、影响等级、应对措施 |

**生成Skill**: `schedule-planner`

---

### 2. 甘特图（Mermaid）

**图表类型**:
| 类型 | 用途 |
|------|------|
| 整体进度甘特图 | 项目总览 |
| 人员泳道甘特图 | 按开发人员分组 |
| 高优先级详细甘特图 | 关键路径分析 |

**生成Skill**: `schedule-planner`

---

### 3. 里程碑计划

**包含内容**:
- 里程碑定义（编号、名称、日期）
- 验收标准清单
- 关联任务列表
- 偏差监控记录

**生成Skill**: `milestone-tracker`

---

### 4. 工时估算表

**包含内容**:
- 功能点清单
- 复杂度评估（高/中/低）
- 三点估算计算
- 缓冲系数应用
- 工时汇总

**生成Skill**: `schedule-planner`

---

### 5. 风险管理清单

**包含内容**:
- 风险识别结果
- 风险等级评估
- 应对措施设计
- 责任人分配
- 状态跟踪

**生成Skill**: `risk-manager`

---

### 6. 项目状态报告

**报告类型**:
| 类型 | 频率 | 内容 |
|------|------|------|
| 日报 | 每日 | 立会摘要、阻塞问题 |
| 周报 | 每周 | 进度统计、偏差分析 |
| 里程碑报告 | 里程碑节点 | 验收报告、经验总结 |
| 发版报告 | 发版节点 | 发版总结、质量数据 |

**生成Skill**: `status-reporter`

---

### 7. 会议纪要

**会议类型**:
| 类型 | 内容 |
|------|------|
| 每日立会 | 三件事汇报、阻塞汇总 |
| 需求评审 | 评审结论、待办事项 |
| 里程碑验收 | 验收检查、结论确认 |
| 变更评审 | 影响分析、审批结论 |

**生成Skill**: `status-reporter`

---

### 8. 变更请求记录

**包含内容**:
- 变换编号
- 变换类型（新增/删除/修改）
- 变换内容描述
- 影响分析结果
- 审批状态

**生成Skill**: `change-handler`

## 协作接口

### 输入
- 功能点清单（来自产品经理）
- 技术方案（来自架构师）
- 工时评估（来自研发）
- 测试排期（来自测试）

### 输出
- 排期表 → 研发、测试、运维
- 状态报告 → 所有干系人
- 里程碑计划 → 项目组全员
- 风险清单 → 项目负责人
