## 七、甘特图Mermaid模板

```mermaid
gantt
    title [项目名称]项目进度
    dateFormat YYYY-MM-DD

    section Sprint1
    [任务名称]           :[task_id], [开始日期], [工期]
    ...

    section Sprint2
    [任务名称]           :[task_id], [开始日期], [工期]
    ...

    section 里程碑
    M1-[里程碑名]        :milestone, m1, [日期], 0d
    M2-[里程碑名]        :milestone, m2, [日期], 0d
```

## 使用说明

1. **排期设计**: 使用模板1-5完成排期设计
2. **文档输出**: 使用模板7-8生成可视化图表
3. **进度跟进**: 使用模板9-10进行日常跟踪
4. **格式选择**: 建议同时输出Markdown+Excel+Mermaid三种格式
