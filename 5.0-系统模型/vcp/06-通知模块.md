# 通知模块 (NTF)

## 功能概述

通知模块负责系统内各类通知的生成、推送和管理，包括服务器邀请、好友请求、@提及通知等，支持桌面端推送和应用内通知。

## 功能清单

| 序号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| 1 | 系统通知 | P0 | 服务器邀请、好友请求等 |
| 2 | 频道内@提及通知 | P0 | 被@时的通知 |
| 3 | 推送通知 | P1 | 桌面端系统推送 |
| 4 | 通知偏好设置 | P1 | 按频道/服务器静音 |

## 详细需求

### 6.1 系统通知

**功能描述：**
系统自动生成各类通知事件。

**通知类型：**
| 类型 | 说明 | 触发条件 |
|------|------|----------|
| SERVER_INVITE | 服务器邀请 | 收到邀请链接 |
| FRIEND_REQUEST | 好友请求 | 收到好友申请 |
| FRIEND_ACCEPT | 好友接受 | 对方接受了好友请求 |
| MENTION | @提及 | 被@或被@角色 |
| ROLE_MENTION | @角色提及 | 被@角色 |
| EVERYONE_MENTION | @全体 | 被@全体成员 |
| DIRECT_MESSAGE | 私聊消息 | 收到私聊（后续） |
| SERVER_BAN | 服务器封禁 | 被服务器封禁 |
| SERVER_KICK | 服务器踢出 | 被服务器踢出 |

**数据表：notification**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| user_id | BIGINT | 是 | 接收者ID |
| type | VARCHAR(50) | 是 | 通知类型 |
| title | VARCHAR(200) | 是 | 通知标题 |
| content | VARCHAR(500) | 否 | 通知内容 |
| source_id | BIGINT | 否 | 来源ID（服务器/频道/消息） |
| source_type | VARCHAR(50) | 否 | 来源类型 |
| is_read | TINYINT | 是 | 是否已读 |
| created_at | DATETIME | 是 | 创建时间 |

### 6.2 频道内@提及通知

**功能描述：**
用户在频道中被@提及或被@角色时，收到应用内通知和推送通知。

**触发场景：**
- 消息中包含 `@用户名`
- 消息中包含 `@角色名`（且用户属于该角色）
- 消息中包含 `@全体成员`

**通知内容：**
- 通知标题：`[服务器名] #频道名`
- 通知内容：`发送者: 消息摘要...`

### 6.3 推送通知

**功能描述：**
应用不在前台时，通过系统推送通知用户。

**推送通道：**
- **桌面端** - Electron的Notification API
- **移动端** - APNs（iOS）/ FCM（Android）（后续排期）

**推送内容：**
- 标题：服务器/频道/发送者信息
- 正文：消息摘要（最多100字符）
- 点击行为：跳转到对应频道/消息

### 6.4 通知偏好设置

**功能描述：**
用户可按服务器和频道自定义通知行为。

**设置层级：**
```
全局设置（系统默认）
    ↓ 继承
服务器设置（覆盖全局）
    ↓ 继承
频道设置（覆盖服务器）
```

**可设置项：**
| 设置项 | 选项 |
|--------|------|
| 通知级别 | 所有消息 / 仅@提及 / 全部静音 |
| 桌面推送 | 开启 / 关闭 |
| 声音提醒 | 开启 / 关闭 |
| 消息预览 | 显示内容 / 仅显示发送者 |
| @提及 | 开启 / 关闭 |

## 接口设计（概要）

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/notification/list | GET | 获取通知列表（分页） |
| /api/notification/{id}/read | POST | 标记单条已读 |
| /api/notification/read-all | POST | 全部标记已读 |
| /api/notification/unread-count | GET | 获取未读数 |
| /api/notification/settings | GET | 获取通知偏好 |
| /api/notification/settings | PUT | 更新通知偏好 |
| /api/server/{id}/notification-setting | PUT | 设置服务器通知级别 |
| /api/channel/{chId}/notification-setting | PUT | 设置频道通知级别 |
