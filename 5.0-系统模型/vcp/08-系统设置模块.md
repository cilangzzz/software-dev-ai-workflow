# 系统设置模块 (STG)

## 功能概述

系统设置模块提供用户个人偏好配置和管理员服务器级配置，包括音频设备选择、快捷键绑定、主题切换、隐私设置等。

## 功能清单

| 序号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| 1 | 个人设置 | P0 | 音频设备、快捷键、主题 |
| 2 | 通知偏好设置 | P1 | 通知行为配置（已含通知模块） |
| 3 | 隐私设置 | P1 | 在线状态、好友请求限制 |
| 4 | 管理员服务器级设置 | P1 | 服务器全局配置 |

## 详细需求

### 8.1 音频设置

**功能描述：**
用户选择和测试音频输入/输出设备。

**设置项：**
| 设置项 | 说明 |
|--------|------|
| 输入设备 | 选择麦克风 |
| 输出设备 | 选择扬声器/耳机 |
| 输入音量 | 麦克风灵敏度（0-100） |
| 输出音量 | 扬声器音量（0-100） |
| 回声消除 | 开关 |
| 噪声抑制 | 开关 |
| 自动增益控制 | 开关 |
| 语音活动检测 | 开关 |
| 静音时阈值 | VAD阈值（0-100） |

**功能：**
- 音频设备列表实时刷新
- 麦克风音量指示条（实时）
- 扬声器测试按钮（播放测试音）

### 8.2 快捷键设置

**功能描述：**
用户自定义快捷键，用于快速操作。

**默认快捷键：**
| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+M | 静音/取消静音 |
| Ctrl+Shift+D | 闭麦/取消闭麦 |
| Ctrl+Shift+S | 连接语音 |
| Ctrl+Shift+L | 断开语音 |
| Ctrl+Shift+P | 推送通话（Push-to-Talk） |

**业务规则：**
- 快捷键支持自定义组合键
- 冲突检测（不能与系统快捷键冲突）
- 支持 Push-to-Talk 模式

### 8.3 主题设置

**功能描述：**
用户切换应用主题风格。

**主题选项：**
- **浅色主题（Light）** - 默认
- **深色主题（Dark）** - 暗色背景
- **跟随系统** - 根据操作系统主题自动切换

**自定义项：**
- 强调色（Accent Color）
- 字体大小（小/中/大）
- 聊天字体大小

### 8.4 隐私设置

**功能描述：**
用户控制自己的隐私相关设置。

**设置项：**
| 设置项 | 选项 | 说明 |
|--------|------|------|
| 在线状态可见范围 | 所有人 / 仅好友 / 无人 | 控制谁能看到在线状态 |
| 好友请求来源 | 所有人 / 服务器成员 / 无人 | 控制谁能发送好友请求 |
| 数据收集 | 启用 / 禁用 | 是否允许收集使用数据 |

### 8.5 管理员服务器级设置

**功能描述：**
服务器管理员配置服务器级别的全局设置。

**设置项：**
| 设置项 | 说明 |
|--------|------|
| 默认通知级别 | 新成员默认的通知设置 |
| 验证等级 | 新成员需要的账号验证等级 |
| 内容过滤 | 是否启用敏感词过滤 |
| 默认语言 | 服务器默认语言（国际化） |
| 成员上限 | 服务器最大成员数 |

**数据表：user_setting**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| user_id | BIGINT | 是 | 用户ID |
| theme | VARCHAR(20) | 是 | 主题：light/dark/system |
| accent_color | VARCHAR(7) | 否 | 强调色 |
| font_size | VARCHAR(10) | 是 | 字体大小：small/medium/large |
| input_device_id | VARCHAR(100) | 否 | 输入设备ID |
| output_device_id | VARCHAR(100) | 否 | 输出设备ID |
| input_volume | INT | 是 | 输入音量（0-100） |
| output_volume | INT | 是 | 输出音量（0-100） |
| echo_cancellation | TINYINT | 是 | 回声消除开关 |
| noise_suppression | TINYINT | 是 | 噪声抑制开关 |
| auto_gain_control | TINYINT | 是 | 自动增益控制开关 |
| push_to_talk | TINYINT | 是 | Push-to-Talk开关 |
| push_to_talk_key | VARCHAR(20) | 否 | Push-to-Talk快捷键 |
| locale | VARCHAR(10) | 是 | 语言设置 |
| online_status_visible | TINYINT | 是 | 在线状态可见范围 |
| friend_request_from | TINYINT | 是 | 好友请求来源 |
| created_at | DATETIME | 是 | 创建时间 |
| updated_at | DATETIME | 是 | 更新时间 |

## 接口设计（概要）

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/settings/audio | GET | 获取音频设置 |
| /api/settings/audio | PUT | 更新音频设置 |
| /api/settings/audio/devices | GET | 获取音频设备列表 |
| /api/settings/audio/test | POST | 测试音频设备 |
| /api/settings/shortcuts | GET | 获取快捷键设置 |
| /api/settings/shortcuts | PUT | 更新快捷键设置 |
| /api/settings/theme | GET | 获取主题设置 |
| /api/settings/theme | PUT | 更新主题设置 |
| /api/settings/privacy | GET | 获取隐私设置 |
| /api/settings/privacy | PUT | 更新隐私设置 |
| /api/settings/export | GET | 导出设置（JSON） |
| /api/settings/import | POST | 导入设置 |

## 国际化

**支持语言：**
- 中文（简体） - 默认
- English

**国际化范围：**
- 应用界面文字
- 系统通知文字
- 错误提示文字
- 日期/时间格式
