# 用户模块 (USR)

## 功能概述

提供用户注册登录、个人资料管理、在线状态管理、好友系统等核心用户功能。

## 功能清单

| 序号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| 1 | 用户注册 | P0 | 手机号/邮箱注册 |
| 2 | 用户登录 | P0 | 账号密码登录、第三方OAuth登录 |
| 3 | 用户资料管理 | P0 | 头像、昵称、个性签名 |
| 4 | 在线状态管理 | P0 | 在线、离开、隐身、请勿打扰 |
| 5 | 好友系统 | P1 | 添加/删除好友、好友列表、好友状态 |

## 详细需求

### 1.1 用户注册

**功能描述：**
用户通过手机号或邮箱注册账号，系统发送验证码完成注册。

**注册方式：**
- 手机号 + 短信验证码
- 邮箱 + 邮件验证码

**业务规则：**
- 同一手机号/邮箱只能注册一个账号
- 用户名长度 2-20 字符，支持中英文、数字、下划线
- 密码长度 8-32 字符，需包含字母和数字
- 注册成功后自动登录

**数据表：user**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| username | VARCHAR(50) | 是 | 用户名（唯一） |
| phone | VARCHAR(20) | 否 | 手机号（唯一） |
| email | VARCHAR(100) | 否 | 邮箱（唯一） |
| password_hash | VARCHAR(255) | 是 | 密码哈希 |
| avatar | VARCHAR(500) | 否 | 头像URL |
| status | TINYINT | 是 | 状态：0-禁用 1-正常 |
| created_at | DATETIME | 是 | 创建时间 |
| updated_at | DATETIME | 是 | 更新时间 |

### 1.2 用户登录

**功能描述：**
支持账号密码登录和第三方OAuth登录。

**登录方式：**
- 用户名/手机号/邮箱 + 密码
- 第三方OAuth（后续排期）

**业务规则：**
- 连续5次密码错误，账号锁定15分钟
- 登录成功返回JWT Token
- Token有效期7天，支持续期

### 1.3 用户资料管理

**功能描述：**
用户可以编辑个人信息，包括头像、昵称、个性签名等。

**可编辑字段：**
- 头像（支持上传，最大2MB，格式：jpg/png/gif）
- 昵称（2-20字符）
- 个性签名（最多100字符）

**数据表：user_profile**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| user_id | BIGINT | 是 | 用户ID（外键） |
| nickname | VARCHAR(50) | 是 | 昵称 |
| avatar | VARCHAR(500) | 否 | 头像URL |
| bio | VARCHAR(200) | 否 | 个性签名 |
| created_at | DATETIME | 是 | 创建时间 |
| updated_at | DATETIME | 是 | 更新时间 |

### 1.4 在线状态管理

**功能描述：**
用户可设置自己的在线状态，其他用户可查看。

**状态类型：**
- **在线（Online）** - 正在使用
- **离开（Away）** - 暂时离开
- **请勿打扰（DND）** - 不接收通知
- **隐身（Invisible）** - 显示为离线

**业务规则：**
- 登录后默认为"在线"
- 关闭客户端后自动设为"离线"
- 状态变更实时同步给好友和同服务器成员

### 1.5 好友系统

**功能描述：**
用户之间可以添加好友，查看好友状态和在线情况。

**功能点：**
- 搜索用户（通过用户名/ID）
- 发送好友请求
- 接受/拒绝好友请求
- 删除好友
- 好友列表展示（按在线状态分组）
- 好友上线提醒

**数据表：user_friend**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| user_id | BIGINT | 是 | 用户ID |
| friend_id | BIGINT | 是 | 好友ID |
| status | TINYINT | 是 | 状态：0-待接受 1-已接受 2-已拒绝 |
| created_at | DATETIME | 是 | 创建时间 |

## 接口设计（概要）

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/auth/register | POST | 用户注册 |
| /api/auth/login | POST | 用户登录 |
| /api/auth/logout | POST | 用户登出 |
| /api/user/profile | GET | 获取个人资料 |
| /api/user/profile | PUT | 更新个人资料 |
| /api/user/avatar | POST | 上传头像 |
| /api/user/status | PUT | 设置在线状态 |
| /api/friend/list | GET | 获取好友列表 |
| /api/friend/request | POST | 发送好友请求 |
| /api/friend/accept | POST | 接受好友请求 |
| /api/friend/reject | POST | 拒绝好友请求 |
| /api/friend/delete | DELETE | 删除好友 |
| /api/friend/search | GET | 搜索用户 |

## 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 用户名已存在 | 提示"用户名已被使用" |
| 手机号/邮箱已注册 | 提示"该手机号/邮箱已注册" |
| 密码错误 | 提示"密码错误"，记录失败次数 |
| 账号锁定 | 提示"账号已锁定，请15分钟后重试" |
| 头像上传超限 | 提示"头像大小不超过2MB" |
| 好友请求重复 | 提示"已发送过好友请求" |

## 验收标准

- [ ] 用户可通过手机号/邮箱注册账号
- [ ] 用户可通过账号密码登录
- [ ] 用户可编辑头像、昵称、个性签名
- [ ] 用户可设置在线状态（在线/离开/DND/隐身）
- [ ] 好友之间可互相查看在线状态
- [ ] 好友请求可发送、接受、拒绝
- [ ] 好友列表按在线状态分组展示
