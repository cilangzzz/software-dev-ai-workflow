# 频道模块 (CHN)

## 功能概述

频道是服务器内的子单元，分为文字频道和语音频道，通过分类（Category）进行组织。用户在频道内进行文字交流或进入语音房间聊天。

## 功能清单

| 序号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| 1 | 频道分类管理 | P0 | 创建/编辑/删除分类，拖拽排序 |
| 2 | 文字频道 | P0 | 创建/编辑/删除文字频道 |
| 3 | 语音频道 | P0 | 创建/编辑/删除语音频道 |
| 4 | 频道权限配置 | P0 | 按角色/成员控制访问权限 |

## 详细需求

### 3.1 频道分类管理

**功能描述：**
分类（Category）用于组织频道，一个分类下可以包含多个文字频道和语音频道。

**业务规则：**
- 分类名称：1-100字符
- 支持拖拽调整分类顺序
- 可折叠/展开分类下的频道
- 删除分类时，其下的频道移至"未分类"

**数据表：channel_category**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| server_id | BIGINT | 是 | 服务器ID |
| name | VARCHAR(100) | 是 | 分类名称 |
| position | INT | 是 | 排序位置 |
| collapsed | TINYINT | 是 | 是否折叠（默认0） |
| created_at | DATETIME | 是 | 创建时间 |

### 3.2 文字频道

**功能描述：**
文字频道用于成员之间的实时文字消息交流。

**业务规则：**
- 频道名称：1-100字符，仅支持小写字母、数字、短横线
- 每个服务器最多100个频道
- 支持置顶消息
- 支持慢速模式（限制发送频率）
- 支持仅管理员发言模式

**数据表：channel**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| server_id | BIGINT | 是 | 服务器ID |
| category_id | BIGINT | 否 | 分类ID（null=未分类） |
| name | VARCHAR(100) | 是 | 频道名称 |
| type | TINYINT | 是 | 类型：1-文字 2-语音 |
| topic | VARCHAR(200) | 否 | 频道主题/描述 |
| position | INT | 是 | 排序位置 |
| slow_mode | INT | 是 | 慢速模式间隔（秒，0=关闭） |
| is_private | TINYINT | 是 | 是否私有频道 |
| status | TINYINT | 是 | 状态：0-归档 1-正常 |
| created_at | DATETIME | 是 | 创建时间 |

### 3.3 语音频道

**功能描述：**
语音频道作为语音房间的入口，用户点击连接后进入语音房间开始语音聊天。

**业务规则：**
- 用户连接语音频道后自动进入语音房间
- 语音频道显示当前在线人数
- 用户上限为20人（可配置）
- 支持设置用户上限
- 支持设置仅管理员可连接

**与语音房间的关系：**
```
语音频道（Channel, type=2）
    ↓ 用户点击"连接"
语音房间（Voice Room）← 实际的语音交互空间
    ↓ 用户离开
返回频道列表
```

### 3.4 频道权限配置

**功能描述：**
频道权限继承自服务器角色权限，可按频道单独覆盖。

**权限项：**
| 权限 | 说明 |
|------|------|
| VIEW_CHANNEL | 查看频道 |
| SEND_MESSAGES | 发送消息（文字频道） |
| MANAGE_MESSAGES | 管理消息 |
| CONNECT | 连接语音频道 |
| SPEAK | 在语音中说话 |
| MUTE_MEMBERS | 静音成员 |
| DEAFEN_MEMBERS | 屏蔽成员音频 |

**继承逻辑：**
```
服务器角色权限（默认）
    ↓ 继承
频道权限覆盖（可单独设置）
    ↓ 合并
最终用户权限 = MAX(角色权限, 频道覆盖权限)
```

**数据表：channel_permission**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | BIGINT | 是 | 主键 |
| channel_id | BIGINT | 是 | 频道ID |
| role_id | BIGINT | 否 | 角色ID（与user_id二选一） |
| user_id | BIGINT | 否 | 用户ID（与role_id二选一） |
| allow_permissions | BIGINT | 是 | 允许的权限位 |
| deny_permissions | BIGINT | 是 | 拒绝的权限位 |
| created_at | DATETIME | 是 | 创建时间 |

## 接口设计（概要）

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/server/{id}/categories | GET | 获取频道分类列表 |
| /api/server/{id}/categories | POST | 创建分类 |
| /api/server/{id}/categories/{catId} | PUT | 编辑分类 |
| /api/server/{id}/categories/{catId} | DELETE | 删除分类 |
| /api/server/{id}/categories/reorder | PUT | 重新排序分类 |
| /api/server/{id}/channels | GET | 获取频道列表 |
| /api/server/{id}/channels | POST | 创建频道 |
| /api/server/{id}/channels/{chId} | GET | 获取频道详情 |
| /api/server/{id}/channels/{chId} | PUT | 编辑频道 |
| /api/server/{id}/channels/{chId} | DELETE | 删除频道 |
| /api/server/{id}/channels/reorder | PUT | 重新排序频道 |
| /api/channel/{chId}/permissions | GET | 获取频道权限 |
| /api/channel/{chId}/permissions | PUT | 设置频道权限 |

## 验收标准

- [ ] 可创建、编辑、删除频道分类
- [ ] 分类支持拖拽排序
- [ ] 可创建文字频道和语音频道
- [ ] 频道支持拖拽排序和跨分类移动
- [ ] 文字频道支持慢速模式
- [ ] 语音频道显示当前在线人数
- [ ] 频道权限可按角色/成员单独配置
- [ ] 权限继承逻辑正确（角色默认→频道覆盖）
