# 质量管理模块需求文档

## 模块概述

| 属性 | 值 |
|------|-----|
| 模块名称 | 质量管理 |
| 英文名称 | Quality Management |
| 优先级 | P1 |
| 所属系统 | 汽车整车装配MES系统 |

## 功能清单

| 功能项 | 功能描述 | 优先级 | 状态 |
|--------|----------|--------|------|
| 检验项目配置 | 配置检验项目和标准 | P1 | 待开发 |
| 过程检验 | 工序过程中的检验记录 | P1 | 待开发 |
| 终检管理 | 整车最终检验 | P1 | 待开发 |
| 不合格处理 | 不合格品处理流程 | P1 | 待开发 |
| 质量追溯 | 质量问题追溯查询 | P0 | 待开发 |

## 数据模型

### 实体定义：MesQualityRecordDO（质量检验记录）

| 字段名 | 字段类型 | 是否必填 | 默认值 | 说明 |
|--------|----------|----------|--------|------|
| id | BIGINT | 是 | 自增 | 主键ID |
| tenant_id | BIGINT | 是 | - | 租户ID |
| vin | VARCHAR(64) | 是 | - | 车辆VIN |
| work_order_id | BIGINT | 否 | - | 工单ID |
| check_type | TINYINT | 是 | - | 检验类型:0-过程检,1-终检 |
| check_item_code | VARCHAR(64) | 是 | - | 检验项目编码 |
| check_item_name | VARCHAR(255) | 是 | - | 检验项目名称 |
| operation_id | BIGINT | 否 | - | 关联工序ID |
| operation_name | VARCHAR(255) | 否 | - | 工序名称 |
| workstation_id | BIGINT | 否 | - | 检验工位ID |
| result | TINYINT | 是 | - | 结果:0-合格,1-不合格 |
| check_value | VARCHAR(255) | 否 | - | 实际检验值 |
| standard_value | VARCHAR(255) | 否 | - | 标准值 |
| inspector_id | BIGINT | 是 | - | 检验员ID |
| inspector_name | VARCHAR(64) | 是 | - | 检验员姓名 |
| check_time | DATETIME | 是 | 当前时间 | 检验时间 |
| defect_reason | VARCHAR(500) | 否 | - | 不合格原因 |
| remark | VARCHAR(500) | 否 | - | 备注 |
| creator | VARCHAR(64) | 是 | - | 创建人 |
| create_time | DATETIME | 是 | 当前时间 | 创建时间 |
| deleted | TINYINT | 是 | 0 | 是否删除 |

### 实体定义：MesDefectHandleDO（不合格处理）

| 字段名 | 字段类型 | 是否必填 | 默认值 | 说明 |

| tenant_id | BIGINT | 是 | - | 租户ID |

| defect_type | VARCHAR(64) | 否 | - | 缺陷类型 |

| handle_status | TINYINT | 是 | 0 | 处理状态:0-待处理,1-处理中,2-已闭环 |
| handler_id | BIGINT | 否 | - | 处理人ID |
| handler_name | VARCHAR(64) | 否 | - | 处理人姓名 |
| handle_time | DATETIME | 否 | - | 处理时间 |

## 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| QC-001 | 终检限制 | 终检不合格的车辆不可入库 |
| QC-002 | 闭环处理 | 不合格问题需闭环处理 |
| QC-003 | 工作流审批 | 关键质量问题需触发工作流审批 |

## 接口设计

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|----------|----------|----------|------|
| 创建检验记录 | POST | /mes/quality/create | 创建检验记录 |
| 检验记录列表 | GET | /mes/quality/page | 分页查询检验记录 |
| 不合格登记 | POST | /mes/defect/create | 登记不合格问题 |
| 不合格处理 | PUT | /mes/defect/handle | 处理不合格问题 |
| 不合格验证 | PUT | /mes/defect/verify | 验证处理结果 |
| 不合格列表 | GET | /mes/defect/page | 分页查询不合格记录 |
