## 一、系统概述

### 1.1 系统简介

汽车整车装配MES系统是面向汽车制造业的生产执行管理系统，实现从工单下发到成品入库的生产全过程管理。系统支持：

- **生产透明化**：实时掌握生产进度、产量、质量等关键指标
- **质量可追溯**：实现车辆VIN追溯和关键零部件追溯
- **作业标准化**：通过工艺路线和作业指导规范化作业流程
- **数据实时化**：设备数据实时采集，异常实时预警

### 1.2 用户角色

| 角色 | 职责 | 主要功能权限 |
|------|------|-------------|
| 生产计划员 | 负责生产计划和工单管理 | 工单创建、下发、监控进度 |
| 车间主任 | 负责车间整体运营管理 | 生产监控、资源调度、异常处理 |
| 班组长 | 负责班组日常管理 | 人员安排、作业指导、问题协调 |
| 作业员 | 产线实际操作人员 | 扫码作业、工序报工、关键件绑定 |
| 质检员 | 负责质量检验 | 过程检验、终检、不合格处理 |
| 仓库管理员 | 负责线边物料管理 | 物料配送、库存管理、缺料预警 |

### 1.3 功能模块概览

| 模块 | 功能描述 | 使用角色 |
|------|---------|---------|
| 生产工单管理 | 创建、下发、跟踪生产工单 | 生产计划员、车间主任 |
| 工艺路线管理 | 定义产品工艺路线和工序 | 生产计划员 |
| 工作站管理 | 管理装配工作站信息 | 车间主任、班组长 |
| 装配作业执行 | 执行装配作业、扫码绑定 | 作业员、班组长 |
| 物料管理 | 线边库存、物料消耗管理 | 仓库管理员 |
| 质量管理 | 质量检验、不合格品处理 | 质检员 |
| 生产追溯 | 产品生产过程追溯查询 | 全员 |
| 生产看板 | 生产状态可视化展示 | 车间主任、班组长 |
| 移动终端 | 移动端扫码作业 | 作业员 |

## 二、快速入门

### 2.1 登录系统

#### PC端登录
1. 打开浏览器，输入系统地址
2. 输入用户名和密码
3. 点击「登录」按钮
4. 首次登录需修改初始密码

#### 移动端登录
1. 打开小程序或APP
2. 输入用户名和密码
3. 点击「登录」按钮
4. 选择当前班次和工作站（如需要）

> **提示**：密码要求8位以上，包含字母和数字。连续5次密码错误将锁定账户30分钟。

### 2.2 首页导航

登录后进入首页，显示以下内容：
- **待办任务**：需要处理的任务列表
- **今日生产**：当日生产进度统计
- **消息通知**：系统通知和异常提醒
- **快捷入口**：常用功能快捷方式

### 2.3 常用操作

#### 扫码开始作业（作业员）
1. 进入「装配作业」模块
2. 扫描工单二维码或输入工单号
3. 确认工序信息
4. 点击「开始作业」

#### 查看生产进度（班组长/车间主任）
1. 进入「生产看板」模块
2. 选择产线和时间段
3. 查看实时产量和进度

## 三、功能模块操作指南

### 3.1 生产工单管理

#### 3.1.1 功能概述
生产工单管理模块用于创建、下发、跟踪生产工单，支持工单的完整生命周期管理。

#### 3.1.2 创建工单

**操作步骤**：
1. 点击「工单管理」→「新建工单」
2. 填写工单信息：
   - 选择产品（必填）
   - 选择产线（必填）
   - 输入计划数量（必填）
   - 选择计划日期（必填）
   - 选择工艺路线（必填）
3. 点击「保存」

**注意事项**：
- 计划数量不能超过产线日产能上限
- 需先配置工艺路线才能创建工单
- 保存后工单状态为「草稿」

#### 3.1.3 下发工单

**操作步骤**：
1. 在工单列表找到草稿状态的工单
2. 点击「下发」按钮
3. 确认下发信息
4. 点击「确认」

**注意事项**：
- 工单下发后不可修改产品信息
- 下发后状态变为「已下发」，产线可开始生产

#### 3.1.4 查看工单进度

**操作步骤**：
1. 在工单列表点击工单号
2. 查看工单详情页
3. 查看「执行进度」区域

**进度信息说明**：
| 字段 | 说明 |
|------|------|
| 计划数量 | 工单计划生产数量 |
| 完成数量 | 已完成生产数量 |
| 合格数量 | 质检合格数量 |
| 进度百分比 | 完成数量/计划数量 |

#### 3.1.5 关闭工单

**适用场景**：工单需要提前终止时

**操作步骤**：
1. 选择需要关闭的工单
2. 点击「关闭」按钮
3. 选择关闭原因
4. 点击「确认」

---

### 3.2 工艺路线管理

#### 3.2.1 功能概述
工艺路线管理模块用于定义产品的装配工序流程，包括工序顺序、工作站分配、标准工时等。

#### 3.2.2 创建工艺路线

**操作步骤**：
1. 点击「工艺路线」→「新建路线」
2. 填写基本信息：
   - 路线名称（必填）
   - 适用产品（必填）
   - 版本号（自动生成）
3. 配置工序：
   - 点击「添加工序」
   - 选择工作站
   - 设置标准工时
   - 设置关键件要求
4. 点击「保存」

#### 3.2.3 配置工序

**工序配置项说明**：

| 配置项 | 说明 |
|--------|------|
| 工序名称 | 工序的名称描述 |
| 工作站 | 执行该工序的工位 |
| 标准工时 | 完成该工序的标准时间（秒） |
| 关键件要求 | 该工序需要绑定的关键件类型 |
| 检验要求 | 是否需要过程检验 |
| 作业指导 | 作业操作说明文档 |

#### 3.2.4 工艺路线生效

**操作步骤**：
1. 在路线列表找到草稿状态的路线
2. 点击「生效」按钮
3. 确认生效

**注意事项**：
- 生效后的工艺路线可用于生产工单
- 已生效的路线不可修改，需创建新版本

### 3.3 装配作业执行

#### 3.3.1 功能概述
装配作业执行模块是作业员日常使用的主要模块，支持扫码作业、工序报工、关键件绑定等功能。

#### 3.3.2 扫码作业流程

**标准作业流程**：
```
扫描VIN码 → 系统识别车辆 → 显示当前工序 → 执行作业 → 扫描关键件 → 完成报工
```

**详细操作步骤**：
1. 进入「装配作业」模块
2. 使用扫描枪扫描车辆VIN码
3. 系统显示车辆信息和当前工序
4. 按作业指导执行作业
5. 如有关键件要求，扫描关键件条码
6. 点击「完成报工」

**VIN码说明**：
- VIN码为17位字符
- 新VIN：系统创建新的作业记录
- 已有VIN：显示作业进度，继续未完成的工序

#### 3.3.3 关键件绑定

**操作步骤**：
1. 在作业界面点击「绑定关键件」
2. 扫描关键件条码
3. 确认关键件信息
4. 点击「确认绑定」

**注意事项**：
- 关键件条码必须与工序要求的关键件类型匹配
- 同一关键件不可重复绑定到其他车辆
- 绑定信息用于质量追溯，请确保准确

#### 3.3.4 异常上报

**适用场景**：作业过程中发现异常情况

**操作步骤**：
1. 在作业界面点击「异常上报」
2. 选择异常类型：
   - 设备故障
   - 物料异常
   - 质量问题
   - 其他异常
3. 填写异常描述
4. 拍照上传（可选）
5. 点击「提交」

---

### 3.4 物料管理

#### 3.4.1 功能概述
物料管理模块用于管理产线工位的线边库存，支持库存查询、物料消耗记录、缺料预警等功能。

#### 3.4.2 库存查询

**操作步骤**：
1. 进入「物料管理」→「库存查询」
2. 选择查询条件：
   - 车间
   - 产线
   - 工位
   - 物料编码/名称
3. 点击「查询」

**库存信息说明**：
| 字段 | 说明 |
|------|------|
| 现有库存 | 当前工位的实际库存数量 |
| 安全库存 | 触发预警的最低库存 |
| 状态 | 正常/预警/缺料 |

#### 3.4.3 物料消耗

**操作步骤**：
1. 进入「物料管理」→「物料消耗」
2. 点击「新增消耗」
3. 选择物料
4. 输入消耗数量
5. 选择关联工单（可选）
6. 点击「保存」

#### 3.4.4 缺料预警

**预警规则**：
- 库存 < 安全库存：黄色预警
- 库存 = 0：红色预警（缺料）

**预警处理**：
1. 收到预警通知后，进入库存查询
2. 确认库存情况
3. 发起配送申请或联系仓库

---

### 3.5 质量管理

#### 3.5.1 功能概述
质量管理模块支持过程检验、终检管理、不合格品处理等质量管理功能。

#### 3.5.2 过程检验

**操作步骤**：
1. 进入「质量管理」→「过程检验」
2. 扫描车辆VIN码
3. 显示检验项目列表
4. 逐项检验并记录结果：
   - 合格：打勾
   - 不合格：填写问题描述
5. 点击「提交」

#### 3.5.3 终检管理

**操作步骤**：
1. 进入「质量管理」→「终检管理」
2. 扫描车辆VIN码
3. 显示完整检验清单
4. 按检验标准逐项检验
5. 填写检验结论：
   - 合格：车辆可入库
   - 不合格：进入不合格处理流程
6. 点击「提交」

#### 3.5.4 不合格处理

**处理流程**：
```
发现不合格 → 记录问题 → 判定处理方式 → 执行处理 → 验证闭环
```

**处理方式**：
- 返修：返回产线重新加工
- 报废：记录报废处理
- 让步接收：经审批后放行

**操作步骤**：
1. 进入「质量管理」→「不合格处理」
2. 找到待处理的记录
3. 填写处理意见
4. 选择处理方式
5. 提交审批（如需要）
6. 执行处理
7. 验证结果并关闭

### 3.6 生产追溯

#### 3.6.1 功能概述
生产追溯模块支持按VIN码、关键件、操作员等维度追溯生产信息。

#### 3.6.2 VIN追溯

**操作步骤**：
1. 进入「生产追溯」→「VIN追溯」
2. 输入或扫描17位VIN码
3. 点击「查询」

**追溯信息包括**：
| 信息类型 | 内容 |
|---------|------|
| 基本信息 | 产品型号、生产日期、班次等 |
| 工序记录 | 各工序完成时间、操作员、工作站 |
| 关键件信息 | 绑定的关键件SN、供应商信息 |
| 质量记录 | 检验记录、不合格处理记录 |
| 设备数据 | 扭矩等设备采集数据 |

#### 3.6.3 关键件追溯

**适用场景**：某关键件发现问题，需要追溯安装到哪些车辆

**操作步骤**：
1. 进入「生产追溯」→「关键件追溯」
2. 输入关键件SN码
3. 点击「查询」
4. 查看该关键件绑定的车辆信息

## 四、常见问题FAQ

### Q1: 忘记密码怎么办？
**A**: 请联系系统管理员重置密码，或使用「忘记密码」功能通过邮箱/手机验证重置。

### Q2: 扫码提示"条码不存在"怎么办？
**A**: 请检查条码格式是否正确，确认条码已在系统中登记。如问题仍存在，请联系系统管理员。

### Q3: 如何修改已完成的作业记录？
**A**: 已完成的作业记录不可直接修改。如需更正，请提交变更申请，经审批后由管理员处理。

### Q4: 关键件绑定错误如何处理？
**A**: 请在发现后及时联系班组长或质检员，由有权限的人员进行解绑操作。

### Q5: 为什么看不到某些功能菜单？
**A**: 系统根据角色权限显示功能菜单。如需访问特定功能，请联系管理员开通相应权限。

### Q6: 移动端无法登录怎么办？
**A**: 请检查网络连接是否正常，确认用户名密码正确。如仍无法登录，请联系技术支持。

### Q7: 工单下发后能否修改？
**A**: 工单下发后，产品、工艺路线等关键信息不可修改。数量可通过「调整」功能修改。

### Q8: 如何查询历史作业记录？
**A**: 进入「装配作业」→「作业记录」，可按时间、VIN码、工单等条件查询。

## 五、附录

### 5.1 术语表

| 术语 | 解释 |
|------|------|
| MES | 制造执行系统（Manufacturing Execution System） |
| VIN | 车辆识别码（Vehicle Identification Number），17位唯一标识 |
| 工单 | 生产任务单，指示生产什么、生产多少、何时生产 |
| 工艺路线 | 产品生产的工序序列和顺序 |
| 关键件 | 影响产品质量和安全的关键零部件 |
| 线边库存 | 产线工位旁的物料库存 |
| 标准工时 | 完成某工序的标准时间 |
| OEE | 设备综合效率（Overall Equipment Effectiveness） |

### 5.2 状态说明

#### 工单状态
| 状态 | 说明 |
|------|------|
| 草稿 | 刚创建，可编辑 |
| 已下发 | 已下发到产线，可开始生产 |
| 生产中 | 正在执行生产 |
| 已完成 | 生产完成 |
| 已关闭 | 提前终止或取消 |

#### 工艺路线状态
| 状态 | 说明 |
|------|------|
| 草稿 | 编辑中，不可用于生产 |
| 已生效 | 可用于生产工单 |
| 已失效 | 不再使用 |
