# 汽车整车装配MES系统 - 快速入门指南

**文档版本：** v1.0
**编写日期：** 2026-03-27

---

## 5分钟快速上手

### 第一步：登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击「登录」

> 提示：首次登录请修改初始密码

### 第二步：熟悉界面

登录后看到首页：
- **顶部导航栏**：功能菜单入口
- **左侧菜单**：具体功能列表
- **中间区域**：工作区和数据展示

### 第三步：开始你的工作

根据你的角色，快速开始：

---

## 作业员 - 扫码作业

```
扫码VIN → 确认工序 → 执行作业 → 扫关键件 → 完成报工
```

**详细步骤**：
1. 点击「装配作业」
2. 用扫描枪扫描车辆VIN码（17位）
3. 查看当前工序和作业指导
4. 完成作业后点击「完成报工」
5. 如有提示，扫描关键件条码

---

## 班组长 - 查看进度

```
选择产线 → 查看看板 → 处理异常
```

**详细步骤**：
1. 点击「生产看板」
2. 选择你的产线
3. 查看实时产量和进度
4. 如有异常提醒，点击处理

---

## 生产计划员 - 创建工单

```
新建工单 → 填写信息 → 下发工单
```

**详细步骤**：
1. 点击「工单管理」→「新建工单」
2. 选择产品、产线
3. 输入计划数量
4. 选择工艺路线
5. 点击「保存」→「下发」

---

## 质检员 - 执行检验

```
扫码车辆 → 逐项检验 → 提交结果
```

**详细步骤**：
1. 点击「质量管理」→「过程检验」
2. 扫描车辆VIN码
3. 按检验项目逐项检查
4. 填写检验结果
5. 点击「提交」

---

## 仓库管理员 - 查看库存

```
选择工位 → 查看库存 → 处理缺料
```

**详细步骤**：
1. 点击「物料管理」→「库存查询」
2. 选择车间/产线/工位
3. 查看库存状态
4. 黄色预警：准备配送
5. 红色缺料：立即配送

---

## 常用快捷操作

| 操作 | 路径 |
|------|------|
| 扫码作业 | 装配作业 → 扫码 |
| 查看进度 | 生产看板 → 产线状态 |
| 查追溯 | 生产追溯 → VIN追溯 |
| 上报异常 | 装配作业 → 异常上报 |

---

## 遇到问题？

| 问题 | 解决方法 |
|------|---------|
| 忘记密码 | 联系管理员重置 |
| 扫码无反应 | 检查扫描枪连接 |
| 看不到菜单 | 确认账号权限 |
| 数据不对 | 刷新页面或重新登录 |
