# 汽车整车装配MES系统 - 数据库设计文档

**文档版本：** v1.0
**创建日期：** 2026-03-24
**作者：** MES架构组

---

## 一、数据库概述

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 多租户支持 | 所有业务表包含tenant_id字段 |
| 逻辑删除 | 使用deleted字段实现软删除 |
| 审计字段 | 统一的creator、create_time、updater、update_time字段 |
| 命名规范 | 表名使用下划线分隔，mes_前缀 |

### 1.2 数据库信息

| 属性 | 值 |
|------|-----|
| 数据库类型 | MySQL 8.x |
| 字符集 | utf8mb4 |
| 排序规则 | utf8mb4_general_ci |
| 存储引擎 | InnoDB |
