# 生产追溯模块需求文档

## 模块概述

| 属性 | 值 |
|------|-----|
| 模块名称 | 生产追溯 |
| 英文名称 | Production Traceability |
| 优先级 | P0 |
| 所属系统 | 汽车整车装配MES系统 |

## 功能清单

| 功能项 | 功能描述 | 优先级 | 状态 |
|--------|----------|--------|------|
| VIN追溯查询 | 按VIN查询车辆生产信息 | P0 | 待开发 |
| 关键件追溯 | 按关键件SN反向追溯车辆 | P0 | 待开发 |
| 工序追溯 | 查询各工序作业详情 | P0 | 待开发 |
| 操作员追溯 | 查询操作员作业记录 | P1 | 待开发 |
| 设备数据追溯 | 查询设备采集数据 | P1 | 待开发 |

### 关键件反向追溯

| 追溯项 | 内容 |
|--------|------|
| 关键件基本信息 | 零部件编码、名称、序列号、供应商 |
| 绑定车辆VIN | 安装在哪辆车上 |
| 绑定时间 | 何时绑定 |
| 绑定工位 | 在哪个工位绑定 |
| 操作员 | 谁执行的绑定操作 |

## 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| TR-001 | 永久保存 | 追溯数据永久保存，不可删除 |
| TR-002 | 权限控制 | 追溯查询需有相应权限 |
| TR-003 | 数据脱敏 | 敏感数据脱敏显示 |

## 接口设计

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|----------|----------|----------|------|
| VIN追溯查询 | GET | /mes/trace/vin | 按VIN查询车辆完整生产信息 |
| 关键件反向追溯 | GET | /mes/trace/part | 按关键件SN追溯车辆 |
| 工序作业详情 | GET | /mes/trace/operation | 查询工序作业详情 |
| 操作员作业记录 | GET | /mes/trace/operator | 查询操作员作业记录 |
| 设备数据追溯 | GET | /mes/trace/device | 查询设备采集数据 |
