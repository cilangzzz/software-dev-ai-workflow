# 物料管理模块需求文档

## 模块概述

| 属性 | 值 |
|------|-----|
| 模块名称 | 物料管理 |
| 英文名称 | Material Management |
| 优先级 | P0 |
| 所属系统 | 汽车整车装配MES系统 |

## 功能清单

| 功能项 | 功能描述 | 优先级 | 状态 |
|--------|----------|--------|------|
| 线边库存查询 | 查询各工位物料库存 | P0 | 待开发 |
| 物料消耗记录 | 记录物料消耗情况 | P0 | 待开发 |
| 缺料预警 | 库存低于安全库存预警 | P0 | 待开发 |
| 物料配送申请 | 申请物料配送 | P1 | 待开发 |
| 物料盘点 | 线边物料盘点 | P1 | 待开发 |

## 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| MT-001 | 自动扣减 | 物料消耗自动扣减线边库存 |
| MT-002 | 缺料预警 | 库存低于安全库存时自动触发预警 |
| MT-003 | 配送审批 | 物料配送申请需审批后执行 |
| MT-004 | 负库存控制 | 库存扣减后不可为负数 |
