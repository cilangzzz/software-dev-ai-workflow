# 工作站管理模块需求文档

## 模块概述

| 属性 | 值 |
|------|-----|
| 模块名称 | 工作站管理 |
| 英文名称 | Workstation Management |
| 优先级 | P0 |
| 所属系统 | 汽车整车装配MES系统 |

## 功能清单

| 功能项 | 功能描述 | 优先级 | 状态 |
|--------|----------|--------|------|
| 工作站创建 | 创建工作站信息 | P0 | 待开发 |
| 工作站编辑 | 修改工作站配置 | P0 | 待开发 |
| 设备绑定 | 绑定工作站关联设备 | P0 | 待开发 |
| 人员配置 | 配置工作站人员 | P1 | 待开发 |
| 工作站状态管理 | 启用/停用工作站 | P0 | 待开发 |

## 工作站类型

| 类型值 | 类型名 | 说明 | 应用场景 |
|--------|--------|------|----------|
| 0 | 普通工位 | 一般装配作业工位 | 零部件安装、线束连接 |
| 1 | 关键工位 | 关键工序作业工位 | 发动机装配、底盘合装 |
| 2 | 检验工位 | 质量检验工位 | 过程检验、终检 |

## 数据模型

### 实体定义：MesWorkstationDO

| 字段名 | 字段类型 | 是否必填 | 默认值 | 说明 |
|--------|----------|----------|--------|------|
| id | BIGINT | 是 | 自增 | 主键ID |
| tenant_id | BIGINT | 是 | - | 租户ID |
| workstation_code | VARCHAR(64) | 是 | - | 工作站编码 |
| workstation_name | VARCHAR(255) | 是 | - | 工作站名称 |
| workshop_id | BIGINT | 否 | - | 车间ID |
| workshop_name | VARCHAR(100) | 否 | - | 车间名称 |
| line_id | BIGINT | 是 | - | 产线ID |
| line_name | VARCHAR(100) | 否 | - | 产线名称 |
| workstation_type | TINYINT | 是 | 0 | 类型:0-普通,1-关键,2-检验 |
| status | TINYINT | 是 | 1 | 状态:0-停用,1-启用 |
| equipment_ids | VARCHAR(500) | 否 | - | 关联设备ID列表(JSON) |
| description | VARCHAR(500) | 否 | - | 描述 |
| creator | VARCHAR(64) | 是 | - | 创建人 |
| create_time | DATETIME | 是 | 当前时间 | 创建时间 |
| updater | VARCHAR(64) | 是 | - | 更新人 |
| update_time | DATETIME | 是 | 当前时间 | 更新时间 |
| deleted | TINYINT | 是 | 0 | 是否删除 |

## 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| WS-001 | 编码唯一 | 工作站编码在同一产线下唯一 |
| WS-002 | 设备多绑定 | 设备可绑定到多个工作站 |
| WS-003 | 停用限制 | 停用的工作站不能分配新工单 |
| WS-004 | 作业检查 | 停用前检查是否有进行中的作业 |

## 接口设计

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|----------|----------|----------|------|
| 创建工作站 | POST | /mes/workstation/create | 创建工作站 |
| 更新工作站 | PUT | /mes/workstation/update | 更新工作站信息 |
| 删除工作站 | DELETE | /mes/workstation/delete | 删除工作站 |
| 获取工作站 | GET | /mes/workstation/get | 获取工作站详情 |
| 工作站列表 | GET | /mes/workstation/page | 分页查询工作站 |
| 按产线查询 | GET | /mes/workstation/list-by-line | 按产线查询工作站 |
| 启用工作站 | PUT | /mes/workstation/enable | 启用工作站 |
| 停用工作站 | PUT | /mes/workstation/disable | 停用工作站 |
| 绑定设备 | PUT | /mes/workstation/bind-equipment | 绑定设备到工作站 |
