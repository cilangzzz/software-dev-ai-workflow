# 生产工单管理模块需求文档

## 模块概述

| 属性 | 值 |
|------|-----|
| 模块名称 | 生产工单管理 |
| 英文名称 | Work Order Management |
| 优先级 | P0 |
| 所属系统 | 汽车整车装配MES系统 |

| 功能项 | 功能描述 | 优先级 | 状态 |
|--------|----------|--------|------|
| 工单创建 | 手动创建或导入生产工单 | P0 | 待开发 |
| 工单编辑 | 修改未下发的工单信息 | P0 | 待开发 |
| 工单下发 | 将工单下发到产线开始执行 | P0 | 待开发 |

## 状态流转

```
┌──────────┐    下发    ┌──────────┐   开始生产   ┌──────────┐   完成    ┌──────────┐
│  待下发  │ ────────> │  已下发  │ ─────────> │  生产中  │ ──────> │  已完成  │
└──────────┘           └──────────┘            └──────────┘          └──────────┘
      │                      │                      │
      │ 关闭                 │ 关闭                 │ 关闭
      ▼                      ▼                      ▼
┌──────────┐           ┌──────────┐           ┌──────────┐
│  已关闭  │           │  已关闭  │           │  已关闭  │
└──────────┘           └──────────┘           └──────────┘
```

### 状态定义

| 状态值 | 状态名 | 说明 | 允许操作 |
|--------|--------|------|----------|
| 0 | 待下发 | 工单已创建，等待下发 | 编辑、下发、关闭 |
| 1 | 已下发 | 工单已下发到产线，等待开始生产 | 开始生产、关闭 |
| 2 | 生产中 | 工单正在生产执行中 | 暂停、完成、关闭 |
| 3 | 已完成 | 工单已完成生产 | 查看、归档 |
| 4 | 已关闭 | 工单被强制关闭 | 查看 |

## 业务规则

| 规则编号 | 规则名称 | 规则描述 |
|----------|----------|----------|
| WO-001 | 编号规则 | 工单编号格式：WO + 年月日 + 4位流水号，如 WO202603240001 |
| WO-002 | 产线唯一 | 同一产线同一时间只能有一个"生产中"状态的工单 |
| WO-003 | 下发锁定 | 工单下发后不可修改产品信息和数量 |
| WO-004 | 完成归档 | 工单完成后自动归档，不可修改 |
| WO-005 | 关联检查 | 关闭工单前检查是否有未完成的作业记录 |

## 用户故事

### US-WO-001: 创建生产工单

**作为** 生产计划员
**我想要** 创建新的生产工单
**以便于** 安排生产任务并下发给车间

**验收标准：**
- Given 用户有创建工单权限
- When 用户填写产品、数量、计划时间等信息并提交
- Then 系统生成工单编号，状态为"待下发"

### US-WO-002: 下发工单

**作为** 生产计划员
**我想要** 下发生产工单到产线
**以便于** 车间开始执行生产任务

**验收标准：**
- Given 工单状态为"待下发"
- When 用户点击下发按钮
- Then 工单状态变为"已下发"，产线收到生产通知

### US-WO-003: 扫码关联工单

**作为** 作业员
**我想要** 扫描VIN码关联到当前工单
**以便于** 记录车辆生产信息

**验收标准：**
- Given 工单状态为"生产中"
- When 作业员扫描VIN码
- Then 系统创建该车辆的作业记录，关联当前工单
