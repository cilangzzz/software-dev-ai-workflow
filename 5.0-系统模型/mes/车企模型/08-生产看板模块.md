# 生产看板模块需求文档

## 模块概述

| 属性 | 值 |
|------|-----|
| 模块名称 | 生产看板 |
| 英文名称 | Production Dashboard |
| 优先级 | P1 |
| 所属系统 | 汽车整车装配MES系统 |

## 功能清单

| 功能项 | 功能描述 | 优先级 | 状态 |
|--------|----------|--------|------|
| 产线状态看板 | 显示产线运行状态 | P1 | 待开发 |
| 实时产量看板 | 实时产量统计 | P1 | 待开发 |
| OEE看板 | 设备综合效率展示 | P2 | 待开发 |
| 异常报警看板 | 异常信息实时展示 | P1 | 待开发 |
| 安灯呼叫看板 | 安灯呼叫信息展示 | P2 | 待开发 |

## 看板展示指标

| 指标 | 说明 | 计算方式 | 更新频率 |
|------|------|----------|----------|
| 计划产量 | 当班计划生产数量 | 工单计划数 | 班次开始时 |
| 实际产量 | 当班实际生产数量 | 完工数统计 | 实时 |
| 差异 | 计划与实际差异 | 计划-实际 | 实时 |
| 合格率 | 一次合格率 | 合格数/检验数 | 实时 |
| 节拍时间 | 实际生产节拍 | 平均单车耗时 | 实时 |
| OEE | 设备综合效率 | 时间稼动率×性能稼动率×良品率 | 每小时 |

## 异常报警类型

| 报警类型 | 说明 | 处理方式 | 级别 |
|----------|------|----------|------|
| 设备故障 | 设备异常停机 | 通知维修 | 高 |
| 质量异常 | 检验不合格 | 通知质量人员 | 高 |
| 缺料预警 | 物料不足 | 通知配送 | 中 |
| 停线报警 | 产线停线 | 通知管理人员 | 高 |

## 接口设计

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|----------|----------|----------|------|
| 产线状态数据 | GET | /mes/dashboard/line-status | 获取产线实时状态数据 |
| 产量数据 | GET | /mes/dashboard/production | 获取产量统计数据 |
| OEE数据 | GET | /mes/dashboard/oee | 获取OEE数据 |
| 异常报警列表 | GET | /mes/dashboard/alerts | 获取异常报警列表 |
| 工位状态列表 | GET | /mes/dashboard/workstation-status | 获取工位状态列表 |
