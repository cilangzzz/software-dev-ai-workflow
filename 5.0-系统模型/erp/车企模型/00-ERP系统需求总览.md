# 汽车制造业ERP系统 - 需求总览

## 项目背景

基于芋道框架(yudao-aisk-mes)SaaS平台，开发汽车制造业ERP系统，支持多租户、多工厂、多组织数据隔离与共享。

| 属性 | 值 |
|------|-----|
| **技术栈** | Spring Boot 2.7 + MyBatis Plus + Redis + MySQL + Flowable |
| **前端** | Vue3 + Element Plus (PC端) / UniApp (移动端) |
| **架构模式** | 单体应用架构 |
| **部署模式** | 多租户SaaS模式 |

## 系统定位

```
┌─────────────────────────────────────────────────────────────┐
│                      ERP计划层                              │
│           (销售、采购、库存、生产计划、财务)                 │
│                    【多租户SaaS平台】                        │
└─────────────────────────────┬───────────────────────────────┘
                              │ 接口对接
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      MES执行层                              │
│              (工单管理、作业执行、质量追溯)                  │
└─────────────────────────────┬───────────────────────────────┘
                              │ 设备集成
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      设备控制层                             │
│                 (PLC、扭矩枪、扫描枪)                       │
└─────────────────────────────────────────────────────────────┘
```

## 实施路线图（财务核算优先）

| 阶段 | 周期 | 内容 | 交付物 |
|------|------|------|--------|
| **Phase 1** | 6周 | 基础模块搭建 | 主数据、系统管理、多租户框架 |
| **Phase 2** | 8周 | 财务核算模块 ★ | 总账、应收、应付、固定资产 |
| **Phase 3** | 6周 | 成本与报表 | 成本管理、财务报表、预算管理 |
| **Phase 4** | 8周 | 供应链模块 | 采购管理、库存管理、供应商管理 |
| **Phase 5** | 6周 | 销售与计划 | 销售管理、生产计划、MES对接 |
| **Phase 6** | 4周 | 扩展模块 | 质量管理、人力资源管理、设备资产 |
| **Phase 7** | 持续 | 优化迭代 | 性能优化、功能完善、运维支持 |
