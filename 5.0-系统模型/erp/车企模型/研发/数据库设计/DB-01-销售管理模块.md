# 销售管理模块(SD)数据库设计文档

## 文档信息
- **文档编号**: DB-01
- **模块名称**: 销售管理模块(Sales and Distribution)
- **版本**: V1.0
- **创建日期**: 2026-03-24
- **数据库类型**: MySQL 8.x

---

## 1. 设计概述

### 1.1 设计原则
- 支持多租户架构，所有业务表包含`tenant_id`字段
- 遵循第三范式，适当冗余提升查询性能
- 采用逻辑删除，保留数据追溯能力
- 统一字段命名规范：下划线命名法(snake_case)

### 1.2 命名规范
| 对象类型 | 命名规则 | 示例 |
|---------|---------|------|
| 表名 | sd_模块_实体 | sd_customer |
| 主键 | id | id |
| 外键 | 关联表_字段 | customer_id |
| 索引 | idx_表名_字段 | idx_sd_order_customer |
| 唯一索引 | uk_表名_字段 | uk_sd_customer_code |

## 2. 表结构设计

### 2.1 客户主数据表 (sd_customer)

**功能说明**: 管理客户基本信息，支持客户分层管理

```sql
CREATE TABLE `sd_customer` (

  `customer_level` TINYINT DEFAULT 3 COMMENT '客户等级:1-A级,2-B级,3-C级',
  `credit_code` VARCHAR(50) DEFAULT NULL COMMENT '统一社会信用代码',
  `industry` VARCHAR(50) DEFAULT NULL COMMENT '所属行业',
  `province` VARCHAR(50) DEFAULT NULL COMMENT '省份',
  `city` VARCHAR(50) DEFAULT NULL COMMENT '城市',
  `address` VARCHAR(500) DEFAULT NULL COMMENT '详细地址',
  `contact_person` VARCHAR(50) DEFAULT NULL COMMENT '联系人',
  `contact_phone` VARCHAR(20) DEFAULT NULL COMMENT '联系电话',
  `contact_email` VARCHAR(100) DEFAULT NULL COMMENT '联系邮箱',
  `sales_rep_id` BIGINT DEFAULT NULL COMMENT '销售代表ID',
  `sales_rep_name` VARCHAR(50) DEFAULT NULL COMMENT '销售代表姓名',
  `credit_limit` DECIMAL(18,2) DEFAULT 0.00 COMMENT '信用额度',
  `credit_used` DECIMAL(18,2) DEFAULT 0.00 COMMENT '已用信用额度',
  `payment_term_id` BIGINT DEFAULT NULL COMMENT '付款条件ID',
  `payment_term_name` VARCHAR(50) DEFAULT NULL COMMENT '付款条件名称',
  `tax_no` VARCHAR(50) DEFAULT NULL COMMENT '税号',
  `bank_name` VARCHAR(100) DEFAULT NULL COMMENT '开户银行',
  `bank_account` VARCHAR(50) DEFAULT NULL COMMENT '银行账号',
  `invoice_type` TINYINT DEFAULT 1 COMMENT '发票类型:1-增值税专用发票,2-增值税普通发票,3-不开票',

  `created_time` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
```

## 4. 索引设计说明

### 4.1 主键索引
所有表均使用自增BIGINT类型主键`id`，支持分布式ID生成策略。

### 4.2 唯一索引
| 表名 | 索引名 | 字段 | 说明 |
|-----|-------|------|------|
| sd_customer | uk_sd_customer_code | tenant_id, customer_code | 租户内客户编码唯一 |
| sd_order | uk_sd_order_no | tenant_id, order_no | 租户内订单号唯一 |
| sd_delivery_plan | uk_sd_delivery_plan_no | tenant_id, plan_no | 租户内计划号唯一 |
| sd_price_list | uk_sd_price_list_code | tenant_id, price_code | 租户内价格表编码唯一 |
| sd_price_item | uk_sd_price_item_product | tenant_id, price_list_id, product_id | 价格表内产品唯一 |
| sd_return_order | uk_sd_return_order_no | tenant_id, return_no | 租户内退货单号唯一 |

### 4.3 普通索引设计原则
1. **租户索引**: 所有表必须包含`tenant_id`索引，支持多租户数据隔离
2. **外键索引**: 所有外键字段建立索引，提升关联查询性能
3. **日期索引**: 业务日期字段建立索引，支持时间范围查询
4. **状态索引**: 状态字段建立组合索引，支持状态筛选

### 4.4 查询优化建议

```sql
-- 订单列表查询优化
SELECT * FROM sd_order
WHERE tenant_id = ?
  AND order_status = ?
  AND order_date BETWEEN ? AND ?
ORDER BY created_time DESC
LIMIT 20;
-- 使用索引: idx_sd_order_status 或 idx_sd_order_date

-- 客户订单汇总查询
SELECT customer_id, customer_name,
       COUNT(*) as order_count,
       SUM(total_amount) as total_amount
FROM sd_order
WHERE tenant_id = ?
  AND order_date BETWEEN ? AND ?
  AND order_status NOT IN (6, 7)
GROUP BY customer_id, customer_name;
-- 使用索引: idx_sd_order_tenant, idx_sd_order_date

-- 价格查询优化
SELECT pi.* FROM sd_price_item pi
INNER JOIN sd_price_list pl ON pi.price_list_id = pl.id
WHERE pl.tenant_id = ?
  AND pi.product_id = ?
  AND pl.status = 1
  AND pl.effective_date <= ?
  AND (pl.expiry_date IS NULL OR pl.expiry_date >= ?)
ORDER BY pl.price_type, pl.effective_date DESC
LIMIT 1;
-- 使用索引: idx_sd_price_item_product, idx_sd_price_list_effective
```

## 6. 多租户设计

### 6.1 租户隔离策略
- 采用共享数据库、共享Schema、租户字段隔离模式
- 所有业务表包含`tenant_id`字段
- 应用层通过TenantContext获取当前租户ID
- MyBatis Plus拦截器自动注入租户条件

### 6.2 租户上下文管理
```java
// 租户上下文工具类
public class TenantContext {
    private static final ThreadLocal<Long> TENANT_ID = new ThreadLocal<>();

    public static void setTenantId(Long tenantId) {
        TENANT_ID.set(tenantId);
    }

    public static Long getTenantId() {
        return TENANT_ID.get();
    }

    public static void clear() {
        TENANT_ID.remove();
    }
}

// MyBatis Plus租户拦截器配置
@Configuration
public class MybatisPlusConfig {
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        TenantLineInnerInterceptor tenantInterceptor = new TenantLineInnerInterceptor();
        tenantInterceptor.setTenantLineHandler(new TenantLineHandler() {
            @Override
            public Expression getTenantId() {
                Long tenantId = TenantContext.getTenantId();
                return new LongValue(tenantId != null ? tenantId : 0);
            }

            @Override
            public String getTenantIdColumn() {
                return "tenant_id";
            }

            @Override
            public boolean ignoreTable(String tableName) {
                // 忽略不需要租户隔离的表
                return false;
            }
        });
        interceptor.addInnerInterceptor(tenantInterceptor);
        return interceptor;
    }
}
```
