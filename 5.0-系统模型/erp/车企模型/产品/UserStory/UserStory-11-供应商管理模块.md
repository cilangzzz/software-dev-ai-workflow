### 2.1 供应商门户

#### US-SP-001 供应商在线注册

| 属性 | 内容 |
|-----|------|
| 故事ID | US-SP-001 |
| 故事名称 | 供应商在线注册 |
| 作为 | 潜在供应商 |
| 我想要 | 通过在线门户注册企业信息并提交认证申请 |
| 以便于 | 成为企业的合格供应商并获得采购订单 |
| 优先级 | P0 |
| 故事点 | 8 |
| 迭代 | Sprint 1 |

**验收标准**

- [ ] GIVEN 我是一个未注册的用户 WHEN 我访问供应商门户首页 THEN 我可以看到"立即注册"按钮
