## 1. 文档概述

### 1.1 文档目的
本文档详细描述财务管理模块的用户故事，从用户视角阐述系统功能和业务价值。

### 1.2 用户角色定义
| 角色 | 说明 | 主要职责 |
|------|------|----------|
| 财务总监(CFO) | 财务部门最高管理者 | 审批重大财务事项、查看财务报表 |
| 财务经理 | 财务部门管理者 | 审核凭证、管理日常财务工作 |
| 总账会计 | 负责总账业务 | 凭证录入、审核、记账、结账 |
| 应收会计 | 负责应收业务 | 销售发票、收款、核销 |
| 应付会计 | 负责应付业务 | 采购发票、付款、核销 |
| 资产会计 | 负责资产业务 | 资产卡片、折旧、处置 |
| 资金专员 | 负责资金业务 | 资金计划、调拨、票据 |
| 成本会计 | 负责成本核算 | 成本计算、分摊、分析 |
| 出纳 | 负责现金业务 | 现金收付、银行对账 |
| 经销商管理员 | 管理经销商返利 | 返利计算、确认、支付 |
| 系统管理员 | 系统维护人员 | 基础设置、权限管理 |

## 2. 总账管理(GL)用户故事

### 2.1 会计科目管理

#### US-GL-001 创建会计科目
**作为** 总账会计
**我想要** 创建和维护会计科目体系
**以便于** 规范企业的财务核算分类

**验收标准**:
- 支持多级科目结构(最多10级)
- 科目编码规则可配置
- 可设置辅助核算项
- 科目属性可选择(资产/负债/权益/成本/损益)
- 科目创建后自动继承上级科目属性
- 提供汽车行业科目模板导入功能

**优先级**: P0
**故事点**: 5

#### US-GL-002 设置辅助核算
**作为** 总账会计
**我想要** 为科目设置辅助核算项
**以便于** 实现多维度财务核算

**验收标准**:
- 支持客户、供应商、部门、项目、成本中心等辅助核算
- 一个科目可设置多个辅助核算项
- 辅助核算项可灵活组合
- 辅助核算余额可单独查询
- 已使用的科目可新增辅助核算项

**优先级**: P0
**故事点**: 3

#### US-GL-003 管理科目模板
**作为** 系统管理员
**我想要** 管理会计科目模板
**以便于** 快速初始化新租户的科目体系

**验收标准**:
- 提供汽车行业标准科目模板
- 支持模板的增删改查
- 模板可复制到具体账套
- 支持Excel导入导出模板
- 模板版本可管理

**优先级**: P1
**故事点**: 3

### 2.2 凭证管理

#### US-GL-004 录入记账凭证
**作为** 总账会计
**我想要** 手工录入记账凭证
**以便于** 记录企业的经济业务

**验收标准**:
- 支持借贷多行分录(无限制行数)
- 凭证号自动生成或手工录入
- 凭证日期自动默认当前日期
- 科目可通过编码或名称模糊搜索
- 支持辅助核算信息录入
- 借贷不平衡时给出提示
- 支持凭证附件上传
- 支持常用摘要选择

**优先级**: P0
**故事点**: 8

#### US-GL-005 使用凭证模板
**作为** 总账会计
**我想要** 使用凭证模板快速生成凭证
**以便于** 提高凭证录入效率

**验收标准**:
- 可创建常用凭证模板
- 模板包含科目、摘要、辅助核算等
- 使用模板时只需填写金额
- 支持模板分类管理
- 模板可共享给其他用户

**优先级**: P1
**故事点**: 3

#### US-GL-006 审核记账凭证
**作为** 财务经理
**我想要** 审核记账凭证
**以便于** 确保凭证的准确性和合规性

**验收标准**:
- 显示待审核凭证列表
- 支持单张或批量审核
- 审核时显示凭证完整信息
- 审核不通过可填写驳回原因
- 审核后凭证状态变更
- 记录审核人和审核时间

**优先级**: P0
**故事点**: 5

#### US-GL-007 记账凭证
**作为** 总账会计
**我想要** 对已审核凭证进行记账
**以便于** 更新账簿余额

**验收标准**:
- 只有已审核凭证可以记账
- 支持单张或批量记账
- 记账时检查凭证状态
- 记账后更新科目余额
- 记账后凭证不可修改
- 记账失败时给出详细错误信息

**优先级**: P0
**故事点**: 5

#### US-GL-008 红冲修改凭证
**作为** 总账会计
**我想要** 通过红冲方式修改已记账凭证
**以便于** 纠正错误的财务记录

**验收标准**:
- 选择原凭证自动生成红冲凭证
- 红冲凭证金额为负数
- 红冲凭证科目与原凭证一致
- 红冲凭证需要审核记账
- 可重新录入正确凭证
- 红冲关系可追溯查询

**优先级**: P0
**故事点**: 5

#### US-GL-009 查询凭证
**作为** 总账会计
**我想要** 按多条件查询凭证
**以便于** 快速找到需要的凭证

**验收标准**:
- 支持按凭证号、日期、科目、金额、摘要等条件查询
- 支持组合条件查询
- 查询结果分页显示
- 支持凭证明细查看
- 支持查询结果导出
- 支持凭证联查原始单据

**优先级**: P0
**故事点**: 3

#### US-GL-010 凭证自动生成
**作为** 总账会计
**我想要** 业务单据自动生成凭证
**以便于** 减少手工录入工作量

**验收标准**:
- 销售出库自动生成应收凭证
- 采购入库自动生成应付凭证

- 费用报销自动生成费用凭证

### 2.4 期末处理

#### US-GL-016 汇兑损益结转
**作为** 总账会计
**我想要** 自动计算并结转汇兑损益
**以便于** 反映外币账户的真实价值

**验收标准**:
- 识别所有外币科目
- 按期末汇率计算汇兑损益
- 自动生成汇兑损益凭证
- 支持预览汇兑损益明细
- 凭证自动审核记账
- 汇率可手工调整

**优先级**: P0
**故事点**: 5

#### US-GL-017 损益结转
**作为** 总账会计
**我想要** 自动结转损益到本年利润
**以便于** 计算本期利润

**验收标准**:
- 自动识别所有损益科目
- 计算损益科目净发生额
- 生成损益结转凭证
- 支持预览结转金额
- 结转后损益科目余额为0
- 凭证自动审核记账

**优先级**: P0
**故事点**: 5

#### US-GL-018 期末结账
**作为** 财务经理
**我想要** 执行期末结账
**以便于** 关闭当前会计期间

**验收标准**:
- 检查所有凭证是否已审核记账
- 检查所有子模块是否已结账
- 检查试算是否平衡
- 检查损益是否已结转
- 结账后期间状态变更为已结账
- 记录结账人和结账时间
- 结账后不可新增凭证

**优先级**: P0
**故事点**: 5

#### US-GL-019 反结账
**作为** 财务总监
**我想要** 对已结账期间进行反结账
**以便于** 纠正结账后的错误

**验收标准**:
- 需要特殊权限
- 只能反结账最后一个已结账期间
- 反结账需要审批流程
- 记录反结账原因和操作人
- 反结账后期间可继续录入凭证
- 后续期间已结账则不允许反结账

**优先级**: P2
**故事点**: 3
