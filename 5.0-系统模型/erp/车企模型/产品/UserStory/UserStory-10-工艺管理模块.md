## 1. 用户角色定义

## 2. 产品管理 (ENG-01) 用户故事

### 2.1 产品主数据管理

#### US-ENG-01-001: 创建产品信息

| 项目 | 内容 |
|------|------|
| **用户故事ID** | US-ENG-01-001 |
| **作为** | 工艺工程师 |
| **我希望** | 能够创建新的产品信息，包括产品编码、名称、规格、分类等基本信息 |
| **以便于** | 建立企业产品主数据库，为后续的BOM管理和生产管理提供基础数据 |
| **优先级** | P1 |
| **故事点** | 5 |

**验收条件:**
- Given 用户已登录系统并具有产品创建权限
- When 用户进入产品管理页面，点击"新增产品"按钮
- Then 系统显示产品信息录入表单
- When 用户填写产品编码(必填)、产品名称(必填)、产品分类(必填)、计量单位(必填)等信息
- And 用户点击"保存"按钮
- Then 系统验证必填字段，保存产品信息，并提示"产品创建成功"
- And 系统自动记录创建人和创建时间

#### US-ENG-01-002: 查询产品信息

| 项目 | 内容 |
|------|------|
| **用户故事ID** | US-ENG-01-002 |
| **作为** | 工艺工程师或生产计划员 |
| **我希望** | 能够通过多种条件查询产品信息 |
| **以便于** | 快速找到所需产品，提高工作效率 |
| **优先级** | P1 |
| **故事点** | 3 |

**验收条件:**
- Given 用户已登录系统
- When 用户进入产品管理页面
- Then 系统显示产品查询区域，支持按产品编码、产品名称、产品分类、产品状态等条件查询
- When 用户输入查询条件并点击"查询"按钮
- Then 系统返回符合条件的产品列表
- And 支持分页显示，默认每页显示20条记录
- And 支持按创建时间、产品编码等字段排序

#### US-ENG-01-003: 修改产品信息

| 项目 | 内容 |
|------|------|

- Then 系统显示产品信息编辑表单，自动填充当前产品信息
- When 用户修改产品信息并点击"保存"按钮
- Then 系统验证修改内容，保存更新，并提示"产品信息更新成功"

#### US-ENG-01-004: 删除产品信息

| 项目 | 内容 |
|------|------|
| **用户故事ID** | US-ENG-01-004 |
| **作为** | 工艺工程师 |
| **我希望** | 能够删除不再使用的产品信息 |
| **以便于** | 保持产品数据库的整洁 |
| **优先级** | P2 |
| **故事点** | 3 |

**验收条件:**
- Given 用户已登录系统并具有产品删除权限
- When 用户在产品列表中点击"删除"按钮
- Then 系统检查产品是否被BOM或其他业务引用
- When 产品未被引用
- Then 系统弹出确认对话框，提示"确定删除该产品吗？"
- When 用户确认删除
- Then 系统执行逻辑删除，将产品状态标记为"已删除"
- And 系统提示"产品删除成功"
- When 产品已被引用
- Then 系统提示"该产品已被使用，无法删除"

## 3. BOM管理 (ENG-02) 用户故事

### 3.1 BOM创建与维护

#### US-ENG-02-001: 创建BOM

| 项目 | 内容 |
|------|------|
| **用户故事ID** | US-ENG-02-001 |
| **作为** | 工艺工程师 |
| **我希望** | 能够创建产品的物料清单(BOM) |
| **以便于** | 定义产品的组成结构，指导生产和采购 |
| **优先级** | P1 |
| **故事点** | 8 |

**验收条件:**
- Given 用户已登录系统并具有BOM创建权限
- And 系统中已存在有效的产品信息
- When 用户进入BOM管理页面，点击"新增BOM"按钮
- Then 系统显示BOM创建界面
- When 用户选择产品(必填)、填写BOM版本(必填)、生效日期(必填)等信息
- And 用户点击"添加物料"按钮，选择子件物料，填写用量、损耗率等
- And 用户继续添加其他子件物料
- And 用户点击"保存"按钮
- Then 系统验证BOM数据完整性，保存BOM信息
- And 系统提示"BOM创建成功"
- And 系统自动计算BOM层级和总物料数量

- And 用户点击"展开"按钮，选择展开方式(单级展开/多级展开/末级展开)

- When 用户选择"多级展开"
- Then 系统显示所有层级的物料组成
- And 每个层级显示物料编码、名称、用量、层级等信息
- And 支持折叠/展开各层级
