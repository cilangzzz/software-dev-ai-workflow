## 1. 模块概述

### 1.1 模块定位
质量管理模块(QM)是汽车制造业ERP系统的核心模块之一,负责构建企业全面质量管理体系,实现从来料检验、过程检验到成品检验的全流程质量管控,支持质量追溯和持续改进,确保产品质量符合行业标准和企业内部要求。

### 1.2 业务价值
- **提升产品质量**: 通过全流程质量管控,降低不良率15-20%
- **降低质量成本**: 减少返工、报废和客诉成本约25%
- **增强追溯能力**: 实现100%质量追溯,满足行业法规要求
- **优化供应链质量**: 来料合格率提升至98%以上
- **支持持续改进**: 建立质量问题闭环管理机制

### 1.3 与MES集成关系
- **ERP职责**: 管理检验标准、不合格处理流程、质量统计分析
- **MES职责**: 执行在线检验、工序检验、实时质量数据采集
- **数据同步**: 质量数据实时同步,实现ERP与MES质量协同

## 2. 业务流程

### 2.1 核心业务流程图

```
来料检验流程:
供应商送货 → 收货登记 → 来料检验申请 → 抽样检验 → 判定 →
合格入库/不合格处理 → 质量记录

过程检验流程:
生产工单 → 首件检验 → 过程巡检 → 完工检验 → 质量数据采集 →
合格转序/不合格处理

成品检验流程:
生产完工 → 成品报检 → 成品检验 → 判定 → 合格入库/不合格处理 →
质量追溯

不合格处理流程:
不合格发现 → 不合格评审 → 处置决策 → 处置执行 → 验证关闭 →
统计分析
```

### 2.2 与其他模块交互关系

| 关联模块 | 交互方式 | 交互内容 |
|---------|---------|---------|
| 采购管理 | 接收数据 | 采购订单、到货信息、供应商信息 |
| 库存管理 | 双向交互 | 检验入库、不合格品隔离、库存质量状态 |
| 生产管理 | 接收数据 | 生产工单、工艺路线、工序信息 |
| 销售管理 | 提供数据 | 质量证明、检验报告、追溯信息 |
| MES系统 | 双向交互 | 检验标准下发、检验数据回传、不合格协同处理 |
| 供应商管理 | 双向交互 | 供应商质量评分、质量索赔 |

## 3. 功能需求

### 3.1 来料检验 (IQC)

#### 3.1.1 来料检验申请
**功能描述**: 支持从来料收货自动生成检验申请,或手工创建检验申请。

**功能要求**:
- 接收采购收货信息自动生成检验申请
- 支持按采购订单、物料批次创建检验申请
- 设置检验类型(常规检验、加严检验、免检)
- 支持检验优先级设置
- 支持批量创建和单据合并

**数据要求**:
- 检验申请单号(自动生成)
- 关联采购订单号/收货单号
- 物料信息(编码、名称、规格、批次)
- 供应商信息
- 送检数量、抽样数量
- 检验标准引用
- 申请时间、申请人

#### 3.1.2 来料检验执行
**功能描述**: 检验员执行来料检验,记录检验结果。

**功能要求**:
- 自动关联检验方案和检验标准
- 支持抽样方案自动计算(AQL标准)
- 检验项目清单展示(尺寸、外观、功能、性能等)
- 支持多种结果录入方式(数值、判定、描述)
- 检验数据自动计算和判定
- 支持检验仪器关联和校准提醒
- 支持检验图片/附件上传
- 检验异常即时上报

**检验类型**:
- **外观检验**: 目视检查项目
- **尺寸检验**: 测量数据记录
- **功能检验**: 功能测试项目
- **性能检验**: 性能指标测试
- **可靠性检验**: 可靠性试验项目

#### 3.1.3 来料检验判定
**功能描述**: 根据检验结果进行判定,支持多种判定结论。

**功能要求**:
- 自动判定规则(基于检验标准)
- 支持人工复判和仲裁
- 判定结果:合格、让步接收、挑选使用、退货
- 生成检验报告
- 支持检验报告电子签名
- 检验结果通知相关人员

**输出物**:
- 来料检验报告
- 不合格报告(如需)
- 让步接收申请(如需)

### 3.2 过程检验 (IPQC)

#### 3.2.1 首件检验
**功能描述**: 生产开始前或变更后进行首件检验,确认生产条件。

**功能要求**:
- 首件检验条件触发(换班、换模、换料等)
- 首件检验项目清单
- 支持快速检验数据录入
- 首件合格/不合格判定
- 首件样品留存管理
- 不合格首件处理流程
- 首件检验记录追溯

**检验时机**:
- 新产品首次生产
- 更换模具/工装
- 更换材料批次
- 设备维修后
- 工艺参数调整后
- 连续停产后复产

#### 3.2.2 过程巡检
**功能描述**: 生产过程中的定期巡检,监控过程质量。

**功能要求**:
- 巡检计划制定(频次、项目)
- 巡检任务自动推送
- 支持移动端巡检验录
- 关键工序重点监控
- 巡检数据实时同步MES
- 异常自动预警
- 巡检SPC分析(可选)

**巡检内容**:
- 产品质量检验
- 工艺参数检查
- 设备状态检查
- 环境条件检查
- 人员操作规范性检查

#### 3.2.3 完工检验
**功能描述**: 工序完工后的检验,决定是否转序或入库。

**功能要求**:
- 完工检验申请(自动/手工)
- 关联生产工单和工序
- 检验项目配置
- 批量抽样检验
- 合格转序/不合格处理
- 检验数据汇总统计
- 与MES完工检验数据协同

#### 3.2.4 过程检验数据采集
**功能描述**: 支持与MES集成,采集过程检验数据。

**功能要求**:
- 接收MES过程检验数据
- 检验数据标准化处理
- 异常数据标识
- 数据完整性校验
- 支持离线数据补录

### 3.3 成品检验 (FQC/OQC)

#### 3.3.1 成品检验申请
**功能描述**: 生产完工产品申请检验。

**功能要求**:
- 从完工报告自动生成检验申请
- 支持按工单、批次检验
- 关联生产过程质量数据
- 检验类型选择(例行检验、型式检验、出货检验)
- 检验优先级管理
- 支持紧急检验通道

#### 3.3.2 成品检验执行
**功能描述**: 成品的全面检验,确保出厂质量。

**功能要求**:
- 成品检验标准配置
- 支持全检和抽检模式
- 检验项目包括:
  - 外观检验
  - 尺寸检验
  - 功能测试
  - 性能测试
  - 安全测试
  - 包装检验
- 检验数据实时记录
- 支持自动测试设备集成
- 检验结果自动判定

#### 3.3.3 出货检验 (OQC)
**功能描述**: 产品出货前的最终检验。

**功能要求**:
- 出货检验申请(关联销售订单)
- 出货检验标准
- 包装完整性检查
- 随机抽样检验
- 出货质量证明生成
- 检验报告打印
- 支持客户特殊检验要求

#### 3.3.4 成品检验报告
**功能要求**:
- 自动生成检验报告
- 支持多种报告模板
- 包含检验数据汇总
- 支持电子签名
- 报告归档管理
- 支持客户定制格式

### 3.4 不合格管理 (NC)

#### 3.4.1 不合格品发现与登记
**功能描述**: 记录各环节发现的不合格品信息。

**功能要求**:
- 多渠道不合格登记(检验、生产、客户投诉)
- 不合格品基本信息录入
- 不合格现象详细描述
- 支持图片/视频上传
- 不合格数量和批次关联
- 不合格来源追溯
- 自动生成不合格报告

**不合格来源**:
- 来料检验不合格
- 过程检验不合格
- 成品检验不合格
- 客户投诉/退货
- 库存盘点发现
- 内部审核发现

#### 3.4.2 不合格评审
**功能描述**: 组织不合格品评审,确定处置方案。

**功能要求**:
- 评审流程发起
- 评审人员指派
- 评审会议记录
- 处置方案建议
  - 返工返修
  - 让步接收
  - 降级使用
  - 报废
  - 退货(供应商)
- 评审结论记录
- 评审费用估算
- 评审报告生成

**评审级别**:
- 一般不合格: 部门级评审
- 严重不合格: 公司级评审
- 重大质量事故: 跨部门评审

#### 3.4.3 不合格处置执行
**功能描述**: 执行不合格品处置方案。

**功能要求**:
- 处置工单生成
- 处置过程记录
- 返工返修作业指导
- 处置后复检安排
- 处置成本统计
- 库存状态更新
- 处置完成确认

#### 3.4.4 不合格品统计分析
**功能要求**:
- 不合格原因分析(柏拉图)
- 不合格趋势分析
- 供应商不合格统计
- 车间/产线不合格统计
- 不合格成本分析
- 8D报告支持

### 3.5 质量追溯

#### 3.5.1 正向追溯
**功能描述**: 从原材料到成品的正向质量追溯。

**功能要求**:
- 按物料批次追溯
- 按生产工单追溯
- 追溯内容包括:
  - 原材料批次信息
  - 来料检验记录
  - 生产过程记录
  - 过程检验记录
  - 成品检验记录
  - 出货记录
- 支持追溯链可视化
- 追溯报告导出

#### 3.5.2 反向追溯
**功能描述**: 从成品/客户投诉追溯到原材料来源。

**功能要求**:
- 按成品批次追溯
- 按销售订单追溯
- 按客户投诉追溯
- 追溯内容包括:
  - 生产过程信息
  - 原材料来源
  - 供应商信息
  - 检验记录
  - 不合格处理记录
- 支持全链条追溯
- 影响范围分析

#### 3.5.3 追溯链管理
**功能要求**:
- 追溯码生成规则
- 追溯关系建立
- 追溯链完整性检查
- 追溯链断裂预警
- 追溯数据归档

#### 3.5.4 质量档案管理
**功能要求**:
- 检验报告归档
- 不合格报告归档
- 质量证明文件管理
- 档案检索查询
- 档案保存期限管理
- 电子档案导出
