## 2. 模块概述

### 2.1 模块定位
财务管理模块是ERP系统的核心枢纽，负责企业所有经济业务的记录、核算、分析和报告。对于汽车制造企业，需要支持经销商返利核算、整车成本核算、研发费用管理和生产成本分摊等行业特色业务。

### 2.2 核心价值
- **财务合规**: 确保财务核算符合企业会计准则和行业规范
- **成本精细**: 支持多维度成本核算，实现整车全生命周期成本管理
- **资金管控**: 实时监控资金流动，优化资金配置效率
- **决策支持**: 提供多角度财务分析，支撑经营决策

### 2.3 子模块划分
| 子模块 | 功能说明 | 优先级 |
|--------|----------|--------|
| 总账管理(GL) | 科目设置、凭证管理、账簿查询、期末结转 | P0 |
| 应收管理(AR) | 客户管理、发票管理、收款管理、账龄分析 | P0 |
| 应付管理(AP) | 供应商管理、发票管理、付款管理、账龄分析 | P0 |
| 固定资产(FA) | 资产卡片、折旧计提、资产变动、资产处置 | P0 |
| 资金管理(TR) | 银行账户、资金计划、资金调拨、票据管理 | P1 |
| 财务报表(FR) | 资产负债表、利润表、现金流量表、成本分析 | P0 |

## 3. 总账管理(GL)

### 3.1 会计科目体系

#### 3.1.1 功能描述
建立多层级会计科目体系，支持科目辅助核算、科目权限控制等功能。

#### 3.1.2 功能列表
| 功能项 | 功能描述 | 优先级 |
|--------|----------|--------|
| 科目设置 | 创建/修改/停用会计科目，支持多级科目结构 | P0 |
| 科目分类 | 资产/负债/权益/成本/损益五大类科目管理 | P0 |
| 辅助核算 | 客户/供应商/部门/项目/成本中心等多维辅助核算 | P0 |
| 科目权限 | 按部门/角色设置科目使用权限 | P1 |
| 科目模板 | 提供汽车行业标准科目模板 | P1 |

#### 3.1.3 业务规则
- 科目编码规则: 4-2-2-2层级结构，如1001.01.01.01
- 科目一旦使用，不可删除，只能停用
- 下级科目的属性继承上级科目，可单独调整
- 辅助核算项必须在科目启用前设置完成

### 3.2 凭证管理

#### 3.2.1 功能描述
管理记账凭证的全生命周期，支持手工录入和系统自动生成。

#### 3.2.2 功能列表
| 功能项 | 功能描述 | 优先级 |
|--------|----------|--------|
| 凭证录入 | 手工录入记账凭证，支持借/贷多行明细 | P0 |
| 凭证模板 | 常用凭证模板管理，快速生成凭证 | P1 |
| 凭证审核 | 三级审核流程（制单-审核-记账） | P0 |
| 凭证查询 | 多条件组合查询凭证 | P0 |
| 凭证修改 | 已审核凭证红冲修改 | P0 |
| 凭证打印 | 标准凭证格式打印输出 | P1 |
| 凭证导入 | Excel批量导入凭证 | P2 |

#### 3.2.3 凭证生成规则
| 业务场景 | 借方科目 | 贷方科目 | 触发条件 |
|----------|----------|----------|----------|
| 销售整车 | 应收账款/银行存款 | 主营业务收入+应交税费 | 销售出库确认 |
| 采购入库 | 原材料/库存商品 | 应付账款/银行存款 | 采购入库确认 |
| 生产领料 | 生产成本 | 原材料 | 领料出库确认 |
| 费用报销 | 管理费用/销售费用 | 银行存款/其他应收款 | 报销单据审批通过 |
| 计提折旧 | 制造费用/管理费用 | 累计折旧 | 月末折旧计提 |
| 经销商返利 | 销售费用 | 应付账款-经销商返利 | 返利确认 |

#### 3.2.4 业务规则
- 凭证号按月连续编号，不允许断号
- 借贷必须平衡，差额为0才能保存
- 已记账凭证不允许直接修改，必须红冲
- 凭证日期必须在当前会计期间内

### 3.3 账簿查询

#### 3.3.1 功能描述
提供各类财务账簿的查询功能，支持多维度数据展示。

#### 3.3.2 账簿类型
| 账簿类型 | 说明 | 查询维度 |
|----------|------|----------|
| 总分类账 | 科目汇总余额查询 | 科目、期间 |
| 明细分类账 | 科目明细发生额查询 | 科目、日期、凭证号 |
| 科目余额表 | 科目期初/本期/期末余额 | 科目、期间、辅助核算 |
| 辅助核算账 | 按辅助项汇总查询 | 辅助核算项、期间 |
| 日记账 | 现金/银行日记账 | 科目、日期 |
| 多栏账 | 多栏式明细账 | 科目、专栏设置 |

### 3.4 期末处理

#### 3.4.1 功能描述
完成会计期间的期末结转和结账处理。

#### 3.4.2 功能列表
| 功能项 | 功能描述 | 执行顺序 |
|--------|----------|----------|
| 汇兑损益结转 | 外币账户汇兑损益计算结转 | 1 |
| 成本结转 | 生产成本结转到库存商品 | 2 |
| 损益结转 | 收入费用结转到本年利润 | 3 |
| 期末调汇 | 期末汇率调整 | 4 |
| 期末结账 | 关闭当前会计期间 | 5 |
| 反结账 | 重新打开已结账期间 | P2 |

#### 3.4.3 结账检查项
- 所有凭证已审核记账
- 所有子模块已结账
- 损益科目余额已结平
- 试算平衡表平衡
- 资产负债表平衡

### 6.2 折旧管理

#### 6.2.1 功能描述
计算和计提固定资产折旧。

#### 6.2.2 功能列表
| 功能项 | 功能描述 | 优先级 |
|--------|----------|--------|
| 折旧计算 | 按折旧方法自动计算折旧 | P0 |
| 折旧计提 | 生成折旧计提凭证 | P0 |
| 折旧调整 | 资产变动后折旧调整 | P0 |
| 折旧查询 | 折旧明细和汇总查询 | P0 |

#### 6.2.3 折旧方法
- **直线法**: 月折旧额 = (原值 - 残值) / 使用月数
- **工作量法**: 单位折旧额 = (原值 - 残值) / 预计总工作量
- **双倍余额递减法**: 年折旧额 = 期初净值 × (2/预计使用年限)
- **年数总和法**: 年折旧额 = (原值 - 残值) × 剩余使用年限 / 年数总和

| 累计折旧调整 | 折旧政策变更 | 借:累计折旧 贷:以前年度损益调整 |
| 使用部门变更 | 资产调拨 | 调整折旧费用归集部门 |
| 使用状态变更 | 在用/停用/封存 | 影响折旧计提 |
| 使用年限调整 | 延长或缩短年限 | 调整后续折旧计算 |

## 9. 汽车行业特色功能

### 9.1 整车成本核算

#### 9.1.1 功能描述
实现整车全生命周期成本核算。

#### 9.1.2 成本构成
| 成本项目 | 说明 | 数据来源 |
|----------|------|----------|
| 直接材料 | 车身、发动机、零部件等 | BOM+采购价 |
| 直接人工 | 生产工人工资 | 工时×工资率 |
| 制造费用 | 生产车间间接费用 | 分摊计算 |
| 模具分摊 | 专用模具费用分摊 | 按产量分摊 |
| 研发分摊 | 车型研发费用分摊 | 按销量分摊 |
| 返利预提 | 预计返利成本 | 按销量预提 |

#### 9.1.3 成本计算流程
1. BOM展开获取材料清单
2. 采购价格获取材料成本
3. 工时统计计算人工成本
4. 制造费用归集和分摊
5. 模具费用按产量分摊
6. 研发费用按销量分摊
7. 返利费用预提计算
8. 汇总生成整车成本

### 9.2 经销商返利核算

#### 9.2.1 功能描述

#### 9.2.2 返利类型

|----------|----------|----------|

| 年度返利 | 年度销量达标奖励 | 现金 |
| 达成返利 | 目标达成奖励 | 现金/抵扣 |
| 建店补贴 | 新建4S店补贴 | 现金 |
| 广告补贴 | 市场推广补贴 | 现金 |
| 金融贴息 | 贷款贴息补贴 | 现金 |

#### 9.2.3 返利处理流程
1. 销量数据采集和汇总
2. 返利规则匹配和计算
3. 返利金额确认和审核
4. 生成返利应付凭证
5. 返利支付申请和审批
6. 返利支付执行

### 9.3 研发费用管理

#### 9.3.1 功能描述

### 10.1 性能需求
| 指标 | 要求 | 说明 |
|------|------|------|

- 操作审计: 关键操作审计日志
- 权限控制: 功能权限和数据权限分离
- 防篡改: 已记账凭证不可修改

- 符合《企业财务会计报告条例》

- 数据保留期限不少于10年

- 与采购模块接口: 采购数据传递

- 与生产模块接口: 生产数据传递

## 11. 约束条件

### 11.1 技术约束
- 后端框架: Spring Boot 2.7
- ORM框架: MyBatis Plus
- 数据库: MySQL 8.x

- 资产编码唯一性
