## 1. 模块概述

### 1.1 模块定位

生产计划模块是汽车制造业ERP系统的核心模块之一，负责统筹企业的生产资源，实现从销售订单到生产执行的全过程计划管理。本模块通过科学的计划编制和优化算法，确保生产资源的最优配置，支持准时制生产(JIT)和准时化顺序供应(JIS)模式，提升生产效率和资源利用率。

### 1.2 业务价值

| 价值维度 | 具体收益 |
|----------|----------|
| 计划准确率 | 提升30%以上，减少紧急插单和计划变更 |
| 库存周转 | 通过JIT拉动降低在制品库存20-30% |
| 产能利用率 | 产能均衡优化，提升设备利用率15-20% |
| 交付准时率 | 提升订单准时交付率至95%以上 |
| 生产效率 | 减少换型时间，提升OEE 10-15% |

### 1.3 业务范围


### 2.2 主生产计划流程

```
1. 需求收集
   ├── 销售订单导入
   ├── 销售预测录入
   ├── 安全库存需求
   └── 补库订单需求

2. 需求平衡
   ├── 需求时栅划分
   ├── ATP/CTP计算
   ├── 需求冲突识别
   └── 协商调整建议

3. 计划编制
   ├── 计划周期设定
   ├── 产品组合规划
   ├── 产量目标分解
   └── 关键资源匹配

4. 粗能力校验
   ├── 关键工作中心识别
   ├── 瓶颈资源分析
   ├── 能力/负荷对比
   └── 调整方案建议

5. 计划发布
   ├── 计划审批流程
   ├── 版本管理
   ├── 计划下达
   └── 相关方通知
```

### 2.3 物料需求计划流程

```
1. MRP运行准备
   ├── MPS数据校验
   ├── BOM数据完整性检查
   ├── 库存数据同步
   └── 参数配置确认

2. MRP计算
   ├── 需求展开(按BOM层级)
   ├── 净需求计算
   │   ├── 毛需求 = 独立需求 + 相关需求
   │   ├── 净需求 = 毛需求 - 现有库存 - 在途 - 已分配
   │   └── 计划订单量 = 净需求 + 安全库存
   ├── 批量策略应用
   └── 提前期倒推

3. 计划订单生成
   ├── 生产订单建议
   ├── 采购申请建议
   ├── 调拨单建议
   └── 异常提示

4. 计划评审
   ├── 采购员确认
   ├── 计划员确认
   ├── 能力再校验
   └── 计划调整

5. 订单下达
   ├── 生产订单下达
   ├── 采购申请提交
   └── 执行进度跟踪
```

### 2.4 生产排程流程

```
1. 排程数据准备
   ├── 计划订单导入
   ├── 工艺路线获取
   ├── 资源可用性检查
   └── 约束条件设定

2. 排程策略配置
   ├── 优先规则设定
   │   ├── 交货期优先
   │   ├── 客户优先级
   │   ├── 工序最短时间
   │   └── 换型时间最小
   ├── 混线比例配置
   └── 资源约束设置

3. 排程计算
   ├── 正向排程(从当前时间开始)
   ├── 逆向排程(从交期倒推)
   ├── 混线平衡优化
   └── 瓶颈资源调度

4. 排程验证
   ├── 能力冲突检测
   ├── 物料齐套检查
   ├── 时间可行性验证
   └── 排程仿真展示

5. 排程发布
   ├── 排程结果审核
   ├── 版本锁定
   ├── 下达MES
   └── 可视化展示
```

## 4. 非功能需求

## 7. 约束条件

### 7.1 业务约束

- 主生产计划一经发布，冻结期内不可随意变更
- MRP运行期间，相关数据需要锁定
- 混线生产必须满足换型约束
- JIS物料必须按顺序供应

### 7.2 技术约束

- 使用Spring Boot 2.7 + MyBatis Plus技术栈
- 数据库为MySQL 8.x
- 支持多租户架构，所有表需包含tenant_id字段
- 前端采用Vue 3.x + Element Plus

## 8. 验收标准

### 8.1 功能验收标准

- 所有P0功能100%实现并通过测试
