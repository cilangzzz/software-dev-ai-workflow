## 1. 模块概述

### 1.1 模块定位

人力资源管理模块(HRM)是汽车制造业ERP系统的核心支撑模块之一，负责管理企业人力资源全生命周期，包括组织架构管理、员工档案管理、考勤管理、薪资管理、绩效管理和培训管理六大子系统。本模块旨在实现人力资源管理的数字化、规范化、智能化，提升人力资源管理效率，支撑企业战略目标实现。

### 1.2 业务价值

- **提升管理效率**: 自动化处理人事事务，减少人工操作，提升HR工作效率50%以上
- **规范管理流程**: 标准化人力资源业务流程，确保合规性和一致性
- **数据驱动决策**: 提供全面的人力资源数据分析，支撑人才战略决策
- **降低用工风险**: 完善的合同管理、考勤管理，降低劳动纠纷风险
- **促进员工发展**: 科学的绩效管理和培训体系，助力员工职业成长

### 1.3 适用范围

本模块适用于汽车制造企业的以下业务场景：
- 集团化多工厂组织架构管理
- 大规模蓝领工人的考勤与排班管理
- 复杂薪资结构(计件工资、加班费、绩效奖金)计算
- 车间班组绩效管理
- 岗位技能培训与认证管理

## 3. 功能需求

### 3.1 组织管理

#### 3.1.1 组织架构管理

**功能描述**:
支持多层级组织架构的建立与维护，包括公司、事业部、部门、班组等层级，支持矩阵式组织结构。

**功能要点**:
- 组织架构树形展示，支持拖拽调整
- 组织单元新增、编辑、删除、合并
- 组织架构版本管理，支持历史版本追溯
- 组织单元属性配置(成本中心、利润中心)
- 虚拟组织管理(项目组、临时团队)
- 组织架构导入导出(Excel模板)

**业务规则**:
- 组织编码唯一，遵循编码规则
- 组织层级不超过8级
- 删除组织需确保无下级组织和关联人员
- 组织变更需记录变更历史

#### 3.1.2 岗位管理

**功能描述**:
管理企业所有岗位信息，建立岗位体系，支撑人员配置和薪酬定级。

**功能要点**:
- 岗位分类管理(管理序列、专业序列、操作序列)
- 岗位职责说明书维护
- 岗位任职资格要求
- 岗位编制管理(定编人数、现有人员、缺编情况)
- 岗位职级体系(P1-P10职级体系)
- 岗位能力模型关联

**业务规则**:
- 岗位编码唯一
- 岗位需关联组织单元
- 岗位编制变更需审批
- 同一组织可有多个相同岗位

#### 3.1.3 编制管理

**功能描述**:
管理人员编制计划，监控编制执行情况，支撑人力成本预算控制。

**功能要点**:
- 年度编制计划制定
- 编制分解(按部门、岗位)
- 编制执行监控(实有人数、在编率)
- 编制变更申请与审批
- 超编预警提醒
- 编制统计报表

**业务规则**:
- 编制计划需经审批后生效
- 超编招聘需特殊审批
- 编制调整需记录原因
- 支持临时编制管理

---

### 3.2 员工档案管理

#### 3.2.1 员工基本信息管理

**功能描述**:
管理员工全量基本信息，建立完整的员工电子档案。

**功能要点**:
- 员工信息录入(基本信息、联系方式、身份信息)
- 员工照片管理(支持人脸识别)
- 紧急联系人信息
- 员工履历信息(教育经历、工作经历)
- 家庭成员信息
- 员工证件管理(身份证、学历证、职业资格证)
- 员工信息变更记录

**业务规则**:
- 工号唯一，遵循编码规则
- 身份证号唯一性校验
- 必填字段完整性校验
- 敏感信息加密存储
- 信息变更需审批

#### 3.2.2 员工入职管理

**功能描述**:
管理员工入职全流程，实现入职办理电子化。

**功能要点**:
- 入职申请发起(招聘系统对接)
- 入职资料收集(证件扫描件上传)
- 入职审批流程
- 入职体检记录
- 劳动合同签订
- 工号自动生成
- 账号自动开通
- 入职指引推送

**业务规则**:
- 入职年龄需满16周岁
- 证件信息与系统信息一致性校验
- 合同签订后方可正式入职
- 入职日期以审批通过日期为准
- 支持批量入职

#### 3.2.3 员工异动管理

**功能描述**:
管理员工在企业的各类人事变动，记录完整的职业轨迹。

**功能要点**:
- 调动管理(部门调动、岗位调动、职级调整)
- 晋升管理(晋升申请、晋升评估、晋升审批)
- 降职管理
- 兼职管理
- 借调管理
- 异动生效时间设置
- 异动历史查询

**业务规则**:
- 异动需发起申请并审批
- 薪资随岗位变动自动调整(可配置)
- 异动生效日期不可早于申请日期
- 同一员工同时只能有一条待生效异动

#### 3.2.4 员工离职管理

**功能描述**:
管理员工离职全流程，确保离职办理规范完整。

**功能要点**:
- 离职申请(员工发起/公司发起)
- 离职面谈记录
- 工作交接管理
- 资产交接管理
- 离职审批流程
- 离职结算(工资、奖金、补偿金)
- 社保公积金停缴
- 账号停用
- 离职证明生成

**业务规则**:
- 试用期离职需提前3天通知
- 正式员工离职需提前30天通知
- 离职需完成工作交接
- 资产未归还不可办理离职
- 离职后档案保留期限不少于2年

#### 3.2.5 劳动合同管理

**功能描述**:
管理员工劳动合同全生命周期，确保用工合规。

**功能要点**:
- 合同签订(首次签订、续签)
- 合同类型管理(固定期限、无固定期限、完成一定任务)
- 合同期限设置
- 合同到期提醒
- 合同续签流程
- 合同变更记录
- 合同解除/终止
- 电子签章集成

**业务规则**:
- 入职30日内必须签订劳动合同
- 试用期包含在合同期限内
- 连续两次固定期限合同后应签无固定期限合同
- 合同到期前30天提醒续签
- 合同变更需双方确认

---

### 3.3 考勤管理

#### 3.3.1 排班管理

**功能描述**:
管理车间班次安排，支持灵活的排班方式。

**功能要点**:
- 班次定义(白班、夜班、轮班)
- 班次时间设置(上班时间、下班时间、休息时间)
- 排班规则配置
- 排班计划制定
- 排班调整申请
- 排班日历展示
- 排班发布与通知
- 倒班管理(四班三倒、三班两倒等)

**业务规则**:
- 班次时间不可重叠
- 排班需提前发布
- 调班需双方同意
- 夜班需满足休息时长要求
- 支持排班模板

#### 3.3.2 考勤打卡

**功能描述**:
记录员工上下班打卡数据，支持多种打卡方式。

**功能要点**:
- 人脸识别打卡(车间考勤机集成)
- 指纹打卡
- 手机APP打卡(外勤人员)
- GPS定位打卡
- WiFi打卡
- 打卡照片记录
- 异常打卡提醒
- 补卡申请

**业务规则**:
- 打卡需在班次时间前后设定范围内
- 漏打卡需当日申请补卡
- 代打卡行为检测
- 外勤打卡需上传现场照片

#### 3.3.3 考勤异常管理

**功能描述**:
处理各类考勤异常情况，确保考勤数据准确。

**功能要点**:
- 迟到记录与统计
- 早退记录与统计
- 漏打卡记录
- 旷工判定与记录
- 异常说明提交
- 异常审批流程
- 异常次数统计
- 异常处罚关联

**业务规则**:
- 迟到/早退时长自动计算
- 连续漏打卡3次记旷工1天
- 旷工自动计入薪资扣除
- 异常审批通过后可修正考勤

#### 3.3.4 请假管理

**功能描述**:
管理员工各类假期申请与审批。

**功能要点**:
- 假期类型管理(年假、病假、事假、婚假、产假、陪产假、丧假等)
- 假期余额管理
- 请假申请
- 请假审批流程
- 请假与排班冲突检测
- 销假管理
- 假期统计报表

**业务规则**:
- 年假按工龄计算(工龄1-10年5天,10-20年10天,20年以上15天)
- 病假需提供医院证明
- 产假98天(难产增加15天,多胞胎每多一婴增加15天)
- 事假不计薪
- 假期余额不足不允许申请

#### 3.3.5 加班管理

**功能描述**:
管理员工加班申请与统计。

**功能要点**:
- 加班申请
- 加班审批流程
- 加班时长统计
- 调休转换
- 加班费计算
- 法定节假日加班识别
- 加班限制设置

**业务规则**:
- 加班需提前申请并审批
- 每日加班不超过3小时,每月不超过36小时
- 工作日加班1.5倍工资
- 周末加班2倍工资
- 法定节假日加班3倍工资
- 加班时长可转换为调休

#### 3.3.6 出差管理

**功能描述**:
管理员工出差申请与报销。

**功能要点**:
- 出差申请
- 出差审批流程
- 行程安排
- 差旅标准管理
- 出差报告
- 与财务报销系统对接

**业务规则**:
- 出差需提前申请
- 出差期间考勤自动标记
- 差旅费用按标准执行

#### 3.3.7 考勤统计报表

**功能描述**:
提供多维度的考勤统计分析。

**功能要点**:
- 日报(当日出勤情况)
- 月报(月度考勤汇总)
- 异常统计(迟到、早退、旷工统计)
- 加班统计
- 请假统计
- 部门考勤对比
- 个人考勤明细
- 考勤数据导出

### 3.4 薪资管理

#### 3.4.1 薪资体系管理

**功能描述**:
建立企业薪酬体系，管理薪资结构。

**功能要点**:
- 薪资等级设置
- 薪资带宽设置(最小值、中位值、最大值)
- 薪资结构定义(基本工资、岗位工资、绩效工资、津贴补贴)
- 薪资项目配置
- 薪资计算公式配置
- 薪资版本管理

**业务规则**:
- 薪资等级与职级对应
- 薪资调整需在带宽范围内
- 新薪资方案需审批后生效

#### 3.4.2 员工薪资管理

**功能描述**:
管理员工薪资信息，支撑薪资核算。

**功能要点**:
- 员工定薪
- 薪资调整(调薪申请、调薪审批、调薪生效)
- 薪资档案
- 薪资历史查询
- 定级定档
- 计件工资设置

**业务规则**:
- 新入职员工定薪需在岗位薪资范围内
- 调薪需审批通过后生效
- 薪资信息保密

#### 3.4.3 薪资核算

**功能描述**:
自动计算员工应发工资，支持多种计薪方式。

**功能要点**:
- 月薪制核算
- 计件制核算(产量*单价)
- 计时制核算
- 加班费计算
- 绩效奖金计算
- 津贴补贴计算
- 社保公积金计算
- 个人所得税计算
- 考勤扣款计算
- 其他扣款(借款、罚款)
- 薪资核算预览
- 薪资核算异常提示

**业务规则**:
- 月薪制: 应发工资 = 基本工资 + 岗位工资 + 绩效工资 + 津贴补贴 - 考勤扣款
- 计件制: 应发工资 = 计件工资 + 津贴补贴 - 扣款
- 加班费 = 加班时数 * 时薪 * 倍率
- 个税按累计预扣法计算
- 社保公积金按当地政策计算

#### 3.4.4 薪资发放

**功能描述**:
管理薪资发放流程，对接银行代发工资。

**功能要点**:
- 工资单生成
- 工资单审核
- 工资单确认
- 银行代发接口
- 发放状态跟踪
- 发放失败处理
- 工资条推送(短信、APP、邮件)
- 电子签章

**业务规则**:
- 工资单需审核确认后发放
- 发放日期统一设置(每月10日/15日)
- 发放失败需重新处理
- 工资条需员工确认签收

#### 3.4.5 社保公积金管理

**功能描述**:
管理员工社保公积金缴纳。

**功能要点**:
- 社保方案配置
- 公积金方案配置
- 缴纳基数设置
- 缴纳比例设置
- 增员减员管理
- 月度缴纳明细
- 社保对账
- 社保转移接续

**业务规则**:
- 入职当月办理增员
- 离职当月办理减员
- 缴纳基数按当地政策
- 支持多地区社保方案

#### 3.4.6 个税管理

**功能描述**:
管理个人所得税计算与申报。

**功能要点**:
- 个税计算(累计预扣法)
- 专项附加扣除管理
- 累计收入统计
- 个税申报数据导出
- 年终奖计税
- 个税汇算清缴

**业务规则**:
- 采用累计预扣预缴法
- 专项附加扣除需员工填报
- 年终奖可选择单独计税

### 3.5 绩效管理

#### 3.5.1 绩效方案管理

**功能描述**:
配置绩效考核方案，支持多种考核模式。

**功能要点**:
- 考核周期设置(月度、季度、年度)
- 考核模板管理
- 考核指标库
- 考核权重配置
- 考核对象设置
- 考核流程配置
- 考核评分规则

**业务规则**:
- 不同岗位类型可配置不同考核方案
- 指标权重之和为100%
- 考核方案变更需审批

#### 3.5.2 KPI指标管理

**功能描述**:
管理关键绩效指标，支撑绩效考核。

**功能要点**:
- KPI指标库建立
- 指标分类(财务、客户、流程、学习成长)
- 指标定义(名称、说明、计算公式)
- 指标目标值设置
- 指标数据来源配置
- 指标分解(公司->部门->个人)
- 指标达成情况跟踪

**业务规则**:
- 指标需量化可衡量
- 指标目标需符合SMART原则
- 指标分解需上下对齐

#### 3.5.3 绩效考核执行

**功能描述**:
执行绩效考核流程，完成绩效评估。

**功能要点**:
- 考核任务发起
- 自评(员工自我评价)
- 上级评价
- 同事评价(360度评估)
- 下级评价(360度评估)
- 评分汇总计算
- 强制分布控制
- 绩效面谈记录
- 考核结果确认

**业务规则**:
- 考核需按时完成
- 评分需有评分说明
- 强制分布:S级10%, A级20%, B级50%, C级15%, D级5%
- 考核结果需员工确认

#### 3.5.4 绩效结果应用

**功能描述**:
将绩效考核结果应用于薪酬调整、晋升、培训等。

**功能要点**:
- 绩效结果统计
- 绩效奖金计算
- 调薪建议
- 晋升降职参考
- 培训需求识别
- 绩效改进计划(PIP)
- 绩效结果归档

**业务规则**:
- S级可晋升或调薪
- 连续2次C级需绩效改进
- 连续2次D级可解除劳动合同
- 绩效结果作为年度评优依据

#### 3.5.5 绩效报表分析

**功能描述**:
提供绩效数据统计与分析。

**功能要点**:
- 部门绩效对比
- 个人绩效排名
- 绩效趋势分析
- 指标达成分析
- 绩效分布统计
- 绩效改进跟踪
- 绩效报表导出

## 5. 业务流程

### 5.1 员工入职流程

```
招聘系统发起入职申请 → HR收集入职资料 → 入职审批 → 体检 → 签订劳动合同
→ 录入系统 → 开通账号 → 分配工位 → 入职培训 → 正式上岗
```

### 5.2 员工离职流程

```
员工提交离职申请 → 部门审批 → 工作交接 → 资产交接 → 财务结算 → HR审批
→ 社保减员 → 账号停用 → 档案归档 → 出具离职证明
```

### 5.3 考勤异常处理流程

```
异常产生 → 员工提交说明 → 班组长审核 → 部门审批 → 考勤修正 → 薪资核算
```

### 5.4 薪资核算流程

```
考勤数据确认 → 绩效数据确认 → 薪资核算 → 核算校验 → 异常处理
→ 工资单生成 → 审批确认 → 银行代发 → 工资条推送
```

### 5.5 绩效考核流程

```
考核方案制定 → 下发考核任务 → 员工自评 → 上级评价 → 评分汇总
→ 强制分布 → 绩效面谈 → 结果确认 → 结果应用
```

### 5.6 培训实施流程

```
培训需求收集 → 培训计划制定 → 培训班创建 → 学员通知 → 培训实施
→ 签到管理 → 效果评估 → 档案归档
```
