## 1. 模块概述

### 1.1 业务背景

成本管理是汽车制造业ERP系统的核心模块之一，负责企业生产经营过程中各项成本的核算、控制、分析和优化。汽车制造业具有成本构成复杂、生产流程长、涉及成本中心多的特点，需要精细化的成本管理工具支撑企业降本增效。

### 1.2 业务价值

- **精准成本核算**: 实现整车成本的精确归集与分配，支持作业成本法(ABC)核算
- **成本控制**: 通过标准成本体系，实现生产过程的成本预警与控制
- **决策支持**: 为产品定价、车型优化、供应链管理提供成本数据支撑
- **利润分析**: 多维度利润分析，支持企业盈利能力评估与改进

### 1.3 目标用户

| 用户角色 | 职责描述 |
|---------|---------|
| 财务总监 | 审批成本政策、查看成本分析报表、成本战略决策 |
| 成本会计 | 成本核算、成本结转、成本报表编制 |
| 成本分析师 | 成本差异分析、成本优化建议 |
| 生产主管 | 车间成本控制、成本中心管理 |
| 采购经理 | 采购成本分析、供应商成本对比 |
| 高层管理 | 成本KPI监控、经营决策支持 |

## 3. 详细功能需求

### 3.1 成本核算子系统

#### 3.1.1 成本要素管理 (F-CO-01-01)

**功能描述**:
成本要素是成本管理的基础数据，用于归集和分类企业的各类成本支出。

**功能要点**:
- 支持成本要素分类管理：材料成本、人工成本、制造费用、管理费用、销售费用等
- 成本要素主数据维护：成本要素编码、名称、类别、会计科目映射
- 成本要素属性设置：直接/间接成本、变动/固定成本、可控/不可控成本
- 成本要素层级管理：支持成本要素组的多层级结构

**数据要求**:
- 成本要素编码唯一性校验
- 成本要素与会计科目的对应关系
- 支持成本要素的启用/停用管理

**业务规则**:
- 成本要素一旦使用不允许删除，只可停用
- 成本要素类别不可修改
- 必须关联会计科目才能用于实际成本核算

#### 3.1.2 成本中心管理 (F-CO-01-02)

**功能描述**:
成本中心是企业内部成本归集和责任划分的基本单元，汽车制造业按车间、班组等设置成本中心。

**功能要点**:
- 成本中心创建与维护：编码、名称、所属部门、负责人
- 成本中心层级结构：支持多层级成本中心树状结构
- 成本中心组管理：按功能、区域等维度建立成本中心组
- 成本中心属性：成本中心类别、产能信息、作业类型
- 成本中心关联：与生产车间、工作中心的关联

**汽车行业特色**:
- 支持冲压、焊装、涂装、总装等工艺车间成本中心
- 支持发动机车间、变速箱车间等零部件成本中心
- 支持班组级成本中心精细化管理

**业务规则**:
- 成本中心必须关联组织架构中的部门
- 成本中心层级不超过5级
- 每个成本中心指定唯一负责人

#### 3.1.3 成本归集 (F-CO-01-03)

**功能描述**:
成本归集是将实际发生的各项成本按照成本中心和成本要素进行收集汇总。

**功能要点**:

**材料成本归集**:
- 从生产领料单自动归集材料成本
- 支持按工单、成本中心归集
- 材料成本差异单独归集
- 支持分批认定法和移动加权平均法

**人工成本归集**:
- 从考勤系统获取工时数据
- 从薪资系统获取人工成本
- 支持直接人工和间接人工分离
- 按成本中心、工单归集人工成本

**制造费用归集**:
- 从财务系统自动获取制造费用
- 支持按费用类别归集
- 折旧费用、能源费用、维修费用等分类归集
- 支持制造费用的预提和待摊处理

**数据来源**:
- 物料管理系统：领料单、退料单
- 生产管理系统：工单、工时记录
- 人事薪资系统：考勤、薪资数据
- 财务系统：费用凭证、固定资产折旧

#### 3.1.4 成本分配 (F-CO-01-04)

**功能描述**:
成本分配是将归集的间接成本按照合理的分配基础分摊到各成本对象。

**功能要点**:

**分配规则设置**:
- 分配基数定义：产量、工时、机器小时、直接材料成本等
- 分配规则配置：支持多级分配、交叉分配
- 分配顺序设置：辅助生产车间->基本生产车间->产品

**辅助生产成本分配**:
- 动力车间成本分配：按用电量分配
- 工具车间成本分配：按工装使用量分配
- 维修车间成本分配：按维修工时分配
- 支持交互分配法、顺序分配法

**制造费用分配**:
- 支持多种分配基准：直接人工工时、机器工时、直接材料成本
- 支持作业成本法分配
- 按车间设置不同的分配率

**业务规则**:
- 分配规则一经确认，会计期间内不得修改
- 分配结果需要审核确认
- 支持分配结果的追溯查询

#### 3.1.5 作业成本核算 (F-CO-01-05)

**功能描述**:
作业成本法(Activity-Based Costing)提供更精确的成本核算方法，适用于汽车制造业复杂的生产流程。

**功能要点**:

**作业定义**:
- 作业库建立：作业编码、作业名称、作业描述
- 作业分类：单位级作业、批次级作业、产品级作业、设施级作业
- 作业中心设置：按车间、工段设置作业中心

**成本动因管理**:
- 成本动因定义：机器小时、检验次数、调整次数、订单数量等
- 动因计量方式：自动采集、手工录入
- 动因数据来源关联

**作业成本计算**:
- 作业成本归集
- 作业成本动因率计算
- 产品/订单作业成本计算

**业务规则**:
- 作业成本核算与传统成本核算可并行
- 支持月度作业成本核算
- 作业成本结果用于管理分析

#### 3.1.6 成本结转 (F-CO-01-06)

**功能要点**:

**生产成本结转**:
- 完工产品成本结转
- 在制品成本保留
- 成本差异结转

**销售成本结转**:
- 销售出库成本结转
- 销售成本与收入匹配
- 支持加权平均、先进先出等计价方法

**成本差异处理**:
- 材料价格差异处理
- 人工效率差异处理
- 制造费用差异处理
- 差异分摊或直接计入损益

### 3.2 标准成本子系统

#### 3.2.1 标准成本制定 (F-CO-02-01)

**功能描述**:
标准成本是成本控制的基准，包括直接材料标准成本、直接人工标准成本、制造费用标准成本。

**功能要点**:

**材料标准成本**:
- 标准单价设置：物料标准采购价格
- 标准用量设置：基于BOM的标准用量
- 材料标准成本 = 标准单价 x 标准用量

**人工标准成本**:
- 标准工资率：按工种、技能等级设置
- 标准工时：基于工艺路线的标准工时
- 人工标准成本 = 标准工资率 x 标准工时

**制造费用标准成本**:
- 变动制造费用标准费率
- 固定制造费用标准费率
- 制造费用标准成本 = 标准费率 x 标准工时

**业务规则**:
- 标准成本需要定期审核和更新
- 标准成本变更需要审批
- 支持标准成本的历史版本管理

#### 3.2.2 BOM成本计算 (F-CO-02-02)

**功能描述**:
基于物料清单(BOM)滚算整车的标准成本。

**功能要点**:

**成本滚算逻辑**:
- 从底层物料开始向上滚算
- 支持多级BOM成本汇总
- 考虑BOM损耗率和废品率
- 支持替代物料成本计算

**整车成本结构**:
- 原材料成本：钢材、铝材、塑料、橡胶等
- 外购件成本：发动机、变速箱、电子设备等
- 自制件成本：冲压件、焊接件等
- 外协件成本：委托加工件

**成本计算要素**:
- 物料成本
- 制造费用分摊
- 外协加工费用

**业务规则**:
- 成本滚算需要完整的BOM和工艺路线
- 支持成本滚算模拟
- 成本变更影响分析

#### 3.2.3 工艺成本计算 (F-CO-02-03)

**功能描述**:
基于工艺路线计算产品的加工成本。

**功能要点**:
- 工序标准工时计算
- 设备标准费率设置
- 工装模具成本分摊
- 支持不同工艺路线的成本计算

#### 3.2.4 成本差异分析 (F-CO-02-05)

**功能描述**:
分析标准成本与实际成本的差异，找出差异原因。

**功能要点**:

**材料成本差异**:
- 材料价格差异 = (实际价格 - 标准价格) x 实际数量
- 材料数量差异 = (实际数量 - 标准数量) x 标准价格

**人工成本差异**:
- 人工工资率差异 = (实际工资率 - 标准工资率) x 实际工时
- 人工效率差异 = (实际工时 - 标准工时) x 标准工资率

**制造费用差异**:
- 变动制造费用差异分析
- 固定制造费用差异分析
- 产能差异分析

**差异原因追溯**:
- 采购价格变动分析
- 生产效率分析
- 质量损失分析
- 工艺变更影响分析

### 3.3 成本分析子系统

#### 3.3.1 成本结构分析 (F-CO-03-01)

**功能描述**:
分析成本的构成要素及其占比。

**功能要点**:

**成本构成分析**:
- 按成本要素分析：材料、人工、制造费用占比
- 按成本中心分析：各车间成本占比
- 按产品分析：各车型成本占比
- 按成本性态分析：变动成本、固定成本占比

**可视化展示**:
- 成本结构饼图
- 成本构成瀑布图
- 成本占比趋势图

**汽车行业特色分析**:
- 整车成本结构分析
- 零部件成本构成分析
- 材料成本占比分析（汽车行业材料成本占比高）

#### 3.3.2 成本趋势分析 (F-CO-03-02)

**功能描述**:
分析成本的历史变化趋势，预测未来成本走向。

**功能要点**:
- 成本趋势图：月度、季度、年度成本趋势
- 同比分析：与去年同期对比
- 环比分析：与上月对比
- 成本预测：基于历史数据预测未来成本

#### 3.3.3 变动固定成本分析 (F-CO-03-03)

**功能描述**:
区分变动成本和固定成本，分析成本性态。

**功能要点**:

**成本性态划分**:
- 变动成本：随产量变动的成本（材料成本、计件工资等）
- 固定成本：不随产量变动的成本（折旧、管理人员工资等）
- 混合成本：半变动成本分解

**盈亏平衡分析**:
- 盈亏平衡点计算
- 安全边际分析
- 经营杠杆分析

**本量利分析**:
- 不同产量水平下的成本分析
- 利润敏感性分析

#### 3.3.4 产品成本分析 (F-CO-03-05)

**功能描述**:
分析各产品的成本情况，支持产品决策。

**功能要点**:
- 各车型成本对比
- 产品成本构成分析
- 产品成本趋势分析
- 产品成本竞争力分析

## 4. 非功能需求

### 4.1 性能需求

| 指标 | 要求 |
|-----|------|
| 成本核算响应时间 | 月末成本核算处理时间 < 2小时 |
| 成本分析报表响应 | 报表查询响应时间 < 3秒 |
| 并发用户数 | 支持至少200个并发用户 |
| 数据处理能力 | 支持百万级成本数据核算 |

### 4.2 安全需求

- 成本数据属于企业核心机密，需要严格的权限控制
- 成本审批流程需要电子签名和审计日志
- 数据传输采用加密传输
- 敏感操作需要二次确认

### 4.3 集成需求

| 集成系统 | 集成方式 | 集成内容 |
|---------|---------|---------|
| 财务系统(FI) | 实时接口 | 成本凭证、科目余额 |
| 生产管理(PM) | 实时接口 | 工单、工时、产量数据 |
| 物料管理(MM) | 实时接口 | 领料、退料、库存数据 |
| 采购管理(PO) | 实时接口 | 采购价格、供应商成本 |
| 销售管理(SD) | 实时接口 | 销售数据、客户信息 |
| 人事系统(HR) | 定时接口 | 薪资、考勤数据 |

### 4.4 数据需求

- 成本数据保留年限：10年
- 成本核算精度：保留2位小数
- 支持多币种成本核算
- 支持多会计准则

## 11. 风险与约束

| 风险 | 影响 | 概率 | 应对措施 |

- 上线时间节点不可推迟
