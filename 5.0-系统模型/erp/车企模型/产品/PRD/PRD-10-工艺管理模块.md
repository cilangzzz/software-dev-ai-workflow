## 1. 模块概述

### 1.1 模块定位

工艺管理模块是汽车制造业ERP系统的核心基础模块，负责管理企业的产品数据、物料清单(BOM)、工艺路线和工程变更。该模块是连接产品研发、生产制造和供应链管理的关键桥梁，确保产品数据的一致性和可追溯性。

### 1.2 业务价值

| 价值维度 | 具体描述 |
|----------|----------|
| 数据一致性 | 建立单一产品数据源，确保设计、采购、生产各环节数据一致 |
| 变更可控 | 规范工程变更流程，降低变更风险，提高变更执行效率 |
| 成本优化 | 精确BOM管理支持成本核算，优化物料配置降低采购成本 |
| 生产效率 | 标准化工艺路线指导生产，提高生产效率和产品质量 |
| 合规追溯 | 完整的产品生命周期记录，满足汽车行业合规要求 |

### 1.3 适用范围

- 整车制造企业
- 汽车零部件生产企业
- 汽车电子制造企业

| ECO名称 | eco_name | VARCHAR(200) | 是 | 变更指令名称 |
| 关联ECR | ecr_id | BIGINT | 是 | 关联ECR ID |
| 执行计划 | execution_plan | TEXT | 是 | 执行计划描述 |
| 计划完成日期 | plan_complete_date | DATE | 是 | 计划完成日期 |
| 实际完成日期 | actual_complete_date | DATE | 否 | 实际完成日期 |
| 责任人 | owner_id | BIGINT | 是 | 变更责任人 |
| 状态 | status | TINYINT | 是 | 1-待执行 2-执行中 3-已完成 4-已关闭 |

#### 3.4.3 变更影响分析

**功能描述**
分析工程变更对产品、BOM、工艺路线、生产计划的影响范围。

**影响分析维度**

| 库存影响 | 影响的库存物料 |

**影响分析报告**

| 报告名称 | 功能描述 |
|----------|----------|
| 变更影响报告 | 汇总变更影响范围 |
| 成本差异报告 | 变更前后的成本对比 |
| 时间影响报告 | 变更对交期的影响分析 |

### 4.2 集成接口

| 接口编号 | 接口名称 | 数据流向 | 功能描述 |
|----------|----------|----------|----------|
| INT-001 | 产品数据同步 | ERP->MES | 同步产品主数据到MES |
| INT-002 | BOM数据同步 | ERP->MES | 同步BOM数据到MES |
| INT-003 | 工艺路线同步 | ERP->MES | 同步工艺路线到MES |
| INT-004 | 工程变更通知 | ERP->MES | 通知MES工程变更 |
| INT-005 | 报工数据回传 | MES->ERP | MES报工数据回传ERP |
| INT-006 | 工序进度同步 | MES->ERP | 同步工序执行进度 |

### 4.3 数据同步规则

| 同步类型 | 触发条件 | 同步内容 | 优先级 |
|----------|----------|----------|--------|
| 实时同步 | BOM/工艺审批通过 | 完整数据立即同步 | P1 |
| 定时同步 | 每日凌晨 | 增量数据同步 | P2 |
| 手动同步 | 用户触发 | 指定数据同步 | P2 |

### 4.4 工程变更同步机制

**变更同步流程**

```
ECO创建 → 变更执行 → 数据更新 → MES通知 → MES确认 → 同步完成
```

**同步状态跟踪**

| 状态 | 说明 |
|------|------|
| 待同步 | 变更数据待同步到MES |
| 同步中 | 数据正在同步 |
| 已同步 | MES已接收数据 |
| 同步失败 | 同步异常需处理 |

## 6. 约束与假设

### 6.1 业务约束

1. 产品编码必须唯一，不可重复
2. BOM版本必须经过审批才能生效
3. 工艺路线必须关联有效的产品
4. 工程变更必须完成审批流程
5. 已使用的BOM版本不可删除，只能作废

### 6.2 技术约束

1. 后端框架：Spring Boot 2.7 + MyBatis Plus
2. 数据库：MySQL 8.x
3. 多租户架构，所有表必须有tenant_id字段
4. 接口规范：RESTful API
5. 认证方式：JWT Token

### 6.3 假设条件

1. 系统已有基础的用户权限管理模块
2. 系统已有基础的物料管理模块
3. 系统已有工作中心管理模块
4. MES系统已部署并具备接口对接能力
