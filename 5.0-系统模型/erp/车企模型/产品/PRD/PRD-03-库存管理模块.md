## 1. 模块概述

### 1.1 模块定位

库存管理模块是汽车制造业ERP系统的核心模块之一，负责企业物料、半成品、成品的全生命周期库存管理。该模块与采购管理、生产管理、销售管理等模块深度集成，实现库存的精准管控、实时追踪和智能预警。

### 1.2 业务背景

汽车制造业具有以下库存管理特点：
- **多仓库复杂管理**：涉及原材料仓、半成品仓、成品仓、线边仓等多种仓库类型
- **VMI供应商库存管理**：支持供应商管理库存模式，降低库存成本
- **批次追溯要求高**：零部件批次追溯满足质量追溯要求
- **整车VIN级管理**：成品车需要精确到VIN码级别的库存管理
- **JIT/JIS配送支持**：支持准时化、顺序化配送需求

### 1.3 目标用户

| 用户角色 | 主要职责 | 核心需求 |
|----------|----------|----------|
| 仓库管理员 | 日常出入库操作、库存盘点 | 操作便捷、实时库存查询 |
| 仓库主管 | 库存监控、预警处理 | 库存分析、预警管理 |
| 采购人员 | 采购计划、供应商协调 | VMI库存查看、补货提醒 |
| 生产计划员 | 生产排程、物料需求 | 物料可用性查询、缺料预警 |
| 财务人员 | 成本核算、库存价值 | 库存成本、呆滞分析 |
| 质量人员 | 质量追溯、批次管理 | 批次追溯、质量锁定 |

## 3. 业务流程

### 3.1 采购入库流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 采购订单到货 │───>│ 质量检验    │───>│ 入库登记    │
└─────────────┘    └─────────────┘    └─────────────┘
                                             │
                                             v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 库存更新完成 │<───│ 生成入库单  │<───│ 库位分配    │
└─────────────┘    └─────────────┘    └─────────────┘
```

**流程说明**：
1. 供应商送货到厂，仓库收货员根据采购订单进行收货
2. 需要检验的物料进入质检区，质检合格后入库
3. 免检物料直接进行入库登记
4. 系统自动分配库位或人工指定库位
5. 生成入库单据，更新库存台账
6. 触发库存预警检查

### 3.2 生产领料流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 生产工单下发 │───>│ 领料申请    │───>│ 领料审批    │
└─────────────┘    └─────────────┘    └─────────────┘
                                             │
                                             v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 库存扣减    │<───│ 出库确认    │<───│ 拣货备料    │
└─────────────┘    └─────────────┘    └─────────────┘
```

**流程说明**：
1. 生产计划下发生产工单
2. 系统自动生成领料申请或人工创建
3. 领料申请进入审批流程
4. 审批通过后，仓库进行拣货备料
5. 生产人员确认领料出库
6. 系统扣减库存，生成出库记录

### 3.3 销售出库流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 销售订单确认 │───>│ 出库申请    │───>│ VIN绑定     │
└─────────────┘    └─────────────┘    └─────────────┘
                                             │
                                             v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 库存扣减    │<───│ 出库确认    │<───│ 拣货备货    │
└─────────────┘    └─────────────┘    └─────────────┘
```

**流程说明**：
1. 销售订单确认后触发出库流程
2. 对于整车，需要绑定具体VIN码
3. 仓库根据出库单进行拣货备货
4. 出库确认后扣减库存
5. 生成出库记录，更新车辆状态

### 3.4 库存盘点流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 盘点计划创建 │───>│ 盘点单生成  │───>│ 实物盘点    │
└─────────────┘    └─────────────┘    └─────────────┘
                                             │
                                             v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 账务处理    │<───│ 差异审批    │<───│ 盘点差异分析│
└─────────────┘    └─────────────┘    └─────────────┘
```

**流程说明**：
1. 创建盘点计划，定义盘点范围和方式
2. 系统生成盘点单，锁定相关库存
3. 仓库人员进行实物盘点
4. 系统对比账面与实物差异
5. 差异审批后进行账务调整
6. 更新库存台账，生成盘点报告

#### 4.2.2 生产入库

**需求描述**：处理生产完工产品的入库业务。

**功能要点**：
- 关联生产工单
- 支持成品、半成品入库
- 支持倒冲入库
- 自动生成VIN码（整车）
- 质检关联

**入库流程**：
1. 生产完工后，生产人员提交入库申请
2. 仓库人员确认入库数量
3. 系统生成入库单据
4. 对于整车，自动或手工绑定VIN码
5. 更新库存台账

#### 4.2.3 VMI入库管理

**需求描述**：支持供应商管理库存模式的入库管理。

**功能要点**：
- VMI供应商库存独立管理
- 支持供应商库存可视化
- 使用结算机制
- 补货预警通知

**VMI库存特点**：
- 物料存放在企业仓库，但所有权归供应商
- 使用后才进行结算和所有权转移
- 需要区分VMI库存和自有库存

### 4.3 出库管理

#### 4.3.1 生产领料出库

**需求描述**：处理生产领料出库业务，支持多种领料模式。

**功能要点**：
- 支持按工单领料、按BOM领料
- 支持领料申请、审批流程
- 支持条码扫描快速出库
- 支持倒冲出库（后进先出）
- 支持批次指定出库

**领料模式**：
| 模式 | 说明 |
|------|------|
| 工单领料 | 根据生产工单领料 |
| 补料领料 | 补充领料 |
| 退料入库 | 剩余物料退库 |
| 倒冲出库 | 生产完工后自动扣减 |

**拣货策略**：
| 策略 | 说明 |
|------|------|
| FIFO | 先进先出 |
| LIFO | 后进先出 |
| FEFO | 先到期先出 |
| 批次指定 | 指定批次出库 |
| 库位优先 | 按库位优先级 |

#### 4.3.2 销售出库

**需求描述**：处理销售订单发货出库业务。

**功能要点**：
- 关联销售订单
- 整车需要绑定VIN码
- 支持部分发货
