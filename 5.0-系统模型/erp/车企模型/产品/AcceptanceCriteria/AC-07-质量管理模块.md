## 1. 来料检验 (IQC) 验收标准

### AC-IQC-001: 来料检验申请功能

#### 功能验收
| 测试场景 | 预期结果 | 验收标准 |
|---------|---------|---------|
| 从收货单创建检验申请 | 自动生成检验申请,关联采购订单和物料信息 | PASS |
| 手工创建检验申请 | 支持录入所有必要信息,生成唯一单号 | PASS |
| 批量创建检验申请 | 支持批量选择收货记录创建申请 | PASS |
| 检验方案自动匹配 | 根据物料自动匹配检验方案和抽样标准 | PASS |
| 检验申请状态流转 | 待检验→检验中→已判定 | PASS |
| 检验申请查询 | 支持多条件查询,结果准确 | PASS |
| 检验申请删除 | 未检验状态可删除,已检验不可删除 | PASS |

#### 数据验收
- [ ] 检验申请单号按规则自动生成,格式: IQC-YYYYMMDD-XXXX
- [ ] 必填字段: 物料编码、批次号、送检数量、供应商
- [ ] 检验方案自动关联,无法手动修改
- [ ] 抽样数量按AQL标准自动计算
- [ ] 申请时间自动记录,精确到秒
- [ ] 数据保存后不可删除检验申请记录

#### 性能验收
- [ ] 单条申请创建响应时间 < 2秒
- [ ] 批量创建100条申请响应时间 < 30秒
- [ ] 申请列表查询响应时间 < 3秒

#### 安全验收
- [ ] 只有授权用户可创建检验申请
- [ ] 操作日志完整记录
- [ ] 数据修改需审批
