# 检查清单：需求就绪

## 检查概要

| 属性 | 值 |

| 阶段 | 需求分析阶段 |

## 必须项 - 全部满足才能进入下一阶段

### 文档维度

- [ ] PRD文档已完成并通过评审

- [ ] 验收标准已定义
- [ ] 需求评审已通过

### 质量维度

- [ ] 用户故事符合INVEST原则
- [ ] 验收标准可测试
- [ ] 需求优先级已确定
- [ ] 干系人已签字确认

### 范围维度

- [ ] 功能需求范围已明确
- [ ] 非功能需求已定义
- [ ] 边界条件已说明

## 条件项 - 根据项目类型选择性满足

### 复杂项目

- [ ] 需求追溯矩阵已完成
- [ ] 业务流程图已绘制
- [ ] 用例图已绘制

### 敏捷项目

- [ ] Sprint Backlog已创建
- [ ] Story Point已估算

## 建议项 - 最佳实践建议

- [ ] 用户旅程图已绘制
- [ ] 竞品分析报告已完成
- [ ] 风险点已识别
