# 流程仪表盘

## 概述

流程仪表盘提供项目流程状态的可视化展示，帮助管理者和团队实时了解项目状态。

## 仪表盘列表

| 看板 | 说明 | 文档 |
|------|------|------|
| 阶段状态看板 | 各阶段当前状态 | [阶段状态看板.md](阶段状态看板.md) |
| 流程效率指标 | 流程效率度量 | [流程效率指标.md](流程效率指标.md) |
| 质量趋势分析 | 质量趋势分析 | [质量趋势分析.md](质量趋势分析.md) |
| 风险预警看板 | 风险预警可视化 | [风险预警看板.md](风险预警看板.md) |

## 使用方式

1. 定期更新仪表盘数据
2. 在周例会/阶段评审中展示
3. 作为项目状态报告的输入
