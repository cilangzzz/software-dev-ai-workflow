### 改进状态定义

| 状态 | 说明 |
|------|------|
| 提议 | 改进建议已提出，待评估 |
| 评估中 | 改进建议正在评估可行性 |
| 已批准 | 改进方案已批准，待实施 |
| 实施中 | 改进方案正在实施 |
| 已完成 | 改进方案已完成实施 |
| 已验证 | 改进效果已验证 |

### 效果评估维度

| 维度 | 评估方法 |
|------|---------|
| 效率提升 | 对比改进前后的流程周期时间 |
| 质量改善 | 对比改进前后的缺陷率、返工率 |
| 满意度 | 收集团队反馈评分 |
