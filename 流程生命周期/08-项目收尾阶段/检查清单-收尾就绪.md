# 检查清单：收尾就绪

## 检查概要

| 属性 | 值 |
|------|-----|
| 阶段 | 项目收尾阶段 |
| 目标 | 项目正式结束 |
| 检查目的 | 确认项目已完成收尾工作 |

## 必须项 - 全部满足项目才算完成

### 验收维度

- [ ] 项目验收报告已完成
- [ ] 干系人已签字确认
- [ ] 验收意见已处理

### 知识维度

- [ ] 项目文档已归档
- [ ] 知识沉淀清单已完成
- [ ] 知识库已更新

### 复盘维度

- [ ] 复盘会议已完成
- [ ] 复盘报告已完成
- [ ] 改进措施已跟踪

### 交接维度

- [ ] 系统权限已交接
- [ ] 运维责任已交接
- [ ] 资源已释放

## 条件项 - 根据项目类型选择性满足

### 长期运营项目

- [ ] 运维团队已接手
- [ ] 运维培训已完成

### 外包项目

- [ ] 验收款已结算
- [ ] 合同已关闭

## 建议项 - 最佳实践建议

- [ ] 团队已表彰
- [ ] 项目成果已分享
- [ ] 客户满意度已调研

## 收尾判定

- [ ] **通过**: 所有必须项已完成，项目可以正式关闭
- [ ] **有条件通过**: 必须项基本完成，有明确的后续计划
- [ ] **不通过**: 存在未完成项，需继续推进
