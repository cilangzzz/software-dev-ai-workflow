## 阶段流转关系

| 前置阶段 | 当前阶段 | 后置阶段 | 交接产出物 |
|---------|---------|---------|-----------|
| 项目立项 | 需求阶段 | 设计阶段 | PRD、用户故事、验收标准 |
| 需求阶段 | 设计阶段 | 开发阶段 | 架构设计、技术选型、数据库设计 |
| 设计阶段 | 开发阶段 | 测试阶段 | 代码、技术文档 |
| 开发阶段 | 测试阶段 | 部署阶段 | 测试报告、验收报告 |
| 测试阶段 | 部署阶段 | 运维阶段 | 部署包、运维手册 |
| - | 运维阶段 | - | 监控报表、容量报告 |
| - | 安全阶段 | - | 安全报告、审计报告 |
| - | 数据阶段 | - | 数据文档、血缘图 |
| 全流程 | 项目管理阶段 | - | 项目计划、状态报告 |

## 质量门控体系

| Gate | 阶段 | 检查项 |
|------|------|--------|
| Gate 1 | 需求阶段 | PRD完整性、用户故事定义、验收标准明确 |
| Gate 2 | 设计阶段 | 设计稿完整性、技术方案评审、数据库设计评审 |
| Gate 3 | 开发阶段 | 代码审查通过、单元测试覆盖率、安全扫描通过 |
| Gate 4 | 测试阶段 | 测试用例覆盖率、缺陷修复率、性能测试通过 |
| Gate 5 | 部署阶段 | 部署方案确认、回滚预案就绪、监控配置完成 |
