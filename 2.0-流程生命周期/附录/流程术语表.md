# 流程术语表

## 术语定义

| 术语 | 英文 | 定义 |
|------|------|------|
| 质量门控 | Gate Check | 阶段间的质量检查点，确保符合标准后才进入下一阶段 |
| 需求追溯矩阵 | RTM | 需求与测试用例的映射关系矩阵 |
| 用户故事 | User Story | 从用户视角描述需求的简短描述 |
| 验收标准 | Acceptance Criteria | 定义需求完成的验收条件 |
| 架构决策记录 | ADR | 记录重要架构决策背景和理由的文档 |
| 技术债务 | Technical Debt | 为短期目标采用的非最优方案，需后续偿还 |
| 返工率 | Rework Rate | 需要返工的产出物比例 |
| 一次通过率 | First Pass Rate | 第一次提交即通过的产出物比例 |
| 缺陷密度 | Defect Density | 单位产出物中发现的缺陷数量 |
| 干系人 | Stakeholder | 与项目有利益关系的个人或组织 |
| 里程碑 | Milestone | 项目关键节点和交付物 |
| 工作分解结构 | WBS | 任务层级分解结构 |
| RACI矩阵 | RACI Matrix | 定义任务角色职责的矩阵 |
| SLA | Service Level Agreement | 服务等级协议 |
| CI/CD | Continuous Integration/Deployment | 持续集成/持续部署 |
| UAT | User Acceptance Test | 用户验收测试 |
