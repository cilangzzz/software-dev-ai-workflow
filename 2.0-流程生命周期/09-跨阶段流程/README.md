# 跨阶段流程

## 概述

跨阶段流程是贯穿项目全生命周期的重要管理流程，不局限于某个特定阶段。

| 流程 | 说明 | 文档 |
|------|------|------|
| 质量门控流程 | 各阶段质量检查标准 | [质量门控流程.md](质量门控流程.md) |
| 变更管理流程 | 全流程变更管理 | [变更管理流程.md](变更管理流程.md) |
| 风险管理流程 | 风险识别与应对 | [风险管理流程.md](风险管理流程.md) |
| 沟通管理流程 | 项目沟通机制 | [沟通管理流程.md](沟通管理流程.md) |
| 供应商管理流程 | 供应商管理 | [供应商管理流程.md](供应商管理流程.md) |
| 合规审计流程 | 合规审计管理 | [合规审计流程.md](合规审计流程.md) |
