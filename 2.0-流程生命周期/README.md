## 流程执行原则

1. **标准化**: 遵循统一的流程规范执行
2. **可追溯**: 所有流程活动可追溯、可审计
3. **自动化**: 优先使用AI Skill辅助流程执行
4. **持续改进**: 定期复盘优化流程

## 快速入门

1. 确定当前项目阶段
2. 查阅对应阶段目录的README
3. 按流程规范执行相关活动
4. 使用检查清单验证阶段就绪
5. 通过质量门控进入下一阶段
