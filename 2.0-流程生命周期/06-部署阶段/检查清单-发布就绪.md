# 检查清单：发布就绪

## 检查概要

| 属性 | 值 |
|------|-----|
| 阶段 | 部署阶段 |
| 目标阶段 | 运维阶段 |
| 检查目的 | 确认系统已具备进入运维阶段的条件 |

## 必须项 - 全部满足才能进入下一阶段

### 部署维度

- [ ] 部署已成功执行
- [ ] 服务正常运行
- [ ] 监控告警已配置

### 验证维度

- [ ] 发布验证已通过
- [ ] 核心功能可用
- [ ] 无严重问题

### 文档维度

- [ ] 运维手册已完成
- [ ] 故障处理手册已完成
- [ ] 发布说明已发布

### 交接维度

- [ ] 运维团队已交接
- [ ] 值班安排已确认
- [ ] 应急联系人已确定

## 条件项 - 根据项目类型选择性满足

### 高可用系统

- [ ] 容灾方案已验证
- [ ] 备份策略已配置
- [ ] 高可用架构已验证

### 数据迁移项目

- [ ] 数据迁移已完成
- [ ] 数据验证已通过

## 建议项 - 最佳实践建议

- [ ] 性能基准已建立
- [ ] 容量规划已完成
- [ ] SLA已定义

## 就绪判定

- [ ] **通过**: 所有必须项已完成，可进入运维阶段
- [ ] **有条件通过**: 必须项基本完成，有明确的补救计划
- [ ] **不通过**: 存在阻塞项，需先解决问题
