# 开发阶段

## 阶段概述

| 属性 | 值 |
|------|-----|
| 阶段名称 | 开发阶段 |
| 阶段编号 | 04 |
| 主要负责部门 | 研发部 |
| 前置条件 | 设计阶段完成 |
| 后置阶段 | 测试阶段 |

## 阶段目标

按照设计文档实现功能代码，保证代码质量和开发效率。

## 关键活动

| 活动 | 说明 | 负责人 |
|------|------|--------|
| 迭代规划 | 规划开发迭代和任务分配 | 技术负责人 |
| 编码开发 | 编写业务代码 | 研发工程师 |
| 代码审查 | 审查代码质量 | 研发工程师 |
| 技术债务管理 | 管理待优化项 | 技术负责人 |

## 产出物

| 产出物 | 类型 | 必要性 |
|--------|------|--------|
| 源代码 | 代码 | 必须 |
| 技术文档 | 文档 | 必须 |
| 代码审查记录 | 文档 | 必须 |
| 开发日志 | 文档 | 建议 |
