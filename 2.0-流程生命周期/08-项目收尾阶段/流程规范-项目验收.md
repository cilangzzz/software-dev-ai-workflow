# 流程规范：项目验收

## 基本信息

| 属性 | 值 |
|------|-----|
| 流程名称 | 项目验收 |
| 所属阶段 | 项目收尾阶段 |
| 流程负责人 | 项目经理 |
| 前置流程 | 运维阶段稳定运行 |
| 后置流程 | 项目复盘 |

## 流程目的

正式验收项目成果，确认项目目标达成。

## 触发条件

- 系统稳定运行一段时间
- 项目目标基本达成

## 执行步骤

### Step 1: 验收准备

- [ ] 整理项目成果
- [ ] 准备验收材料
- [ ] 制定验收标准

### Step 2: 验收评审

- [ ] 组织验收会议
- [ ] 演示项目成果
- [ ] 答疑和讨论
- [ ] 记录验收意见

### Step 3: 验收确认

- [ ] 整改验收意见
- [ ] 获取验收签字
- [ ] 归档验收材料

## 输出物

| 输出物 | 格式 | 存放位置 |
|--------|------|---------|
| 项目验收报告 | 文档 | 项目文档库 |
| 验收确认书 | 文档 | 项目文档库 |

## 质量标准

- 验收标准全部达成
- 干系人已签字确认
- 文档已归档
