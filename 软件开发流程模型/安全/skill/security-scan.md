# Skill: security-scan

## 基本信息
- **名称**: security-scan
- **版本**: 1.0.0
- **所属部门**: 安全部
- **优先级**: P0

## 功能描述
集成多种安全扫描工具，执行全面的安全扫描。包括SAST（静态分析）、SCA（依赖扫描）、密钥检测、容器扫描等。

## 触发条件
- 命令触发: `/security-scan`
- 自然语言触发:
  - "执行安全扫描"
  - "全面安全检查"
  - "安全扫描"

## 输入参数
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| target | string | 是 | 扫描目标：代码目录/镜像/依赖文件 |
| scan_types | string | 否 | 扫描类型：sast / sca / secrets / container / all |
| fail_threshold | string | 否 | 失败阈值：critical / high / medium |

## 执行流程
1. **预扫描** - 快速识别可疑区域
2. **深度扫描** - 执行各类安全扫描
3. **结果聚合** - 合并和去重扫描结果
4. **优先级排序** - 按风险排序
5. **报告生成** - 生成统一报告

## 扫描类型
| 类型 | 说明 | 工具示例 |
|------|------|---------|
| SAST | 静态代码安全分析 | Semgrep, CodeQL |
| SCA | 第三方依赖漏洞 | Trivy, Snyk |
| Secrets | 密钥和凭据检测 | Gitleaks, TruffleHog |
| Container | 容器镜像扫描 | Trivy, Grype |

scan_types: all
