# Skill: data-lineage-trace

## 基本信息
- **名称**: data-lineage-trace
- **版本**: 1.0.0
- **所属部门**: 数据部
- **优先级**: P0

## 功能描述
追踪数据血缘关系，分析数据的来源和去向。支持影响分析和根因定位。

## 触发条件
- 命令触发: `/data-lineage-trace`
- 自然语言触发:
  - "追踪数据血缘"
  - "分析数据来源"
  - "影响分析"

## 输入参数
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| target | string | 是 | 目标表/字段/指标 |
| direction | string | 否 | 追踪方向：upstream / downstream / both |
| depth | number | 否 | 追踪深度，默认3层 |

## 执行流程
1. **目标识别** - 解析追踪目标
2. **血缘解析** - 解析SQL/代码获取血缘关系
3. **关系构建** - 构建血缘关系图
4. **可视化输出** - 生成血缘图谱
5. **影响分析** - 分析变更影响范围
