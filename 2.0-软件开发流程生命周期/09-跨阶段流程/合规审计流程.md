# 合规审计流程

## 流程概述

合规审计流程确保项目符合法律法规、行业标准和公司政策要求。

## 合规类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 安全合规 | 信息安全相关要求 | 处理敏感数据的系统 |
| 隐私合规 | 数据隐私保护要求 | 涉及个人信息的项目 |
| 行业合规 | 行业监管要求 | 金融、医疗等行业 |
| 内部合规 | 公司内部政策要求 | 所有项目 |

## 审计流程

```
┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│  合规准备  │──▶│  合规检查  │──▶│  问题整改  │──▶│  复审验证  │──▶│  合规认证  │
└──────────┘   └──────────┘   └──────────┘   └──────────┘   └──────────┘
```

## 执行步骤

### Step 1: 合规准备

- [ ] 识别合规要求
- [ ] 准备合规材料
- [ ] 自检合规状态

### Step 2: 合规检查

- [ ] 组织合规审计
- [ ] 检查合规材料
- [ ] 记录检查结果

### Step 3: 问题整改

- [ ] 整理问题清单
- [ ] 制定整改计划
- [ ] 执行整改措施

### Step 4: 复审验证

- [ ] 验证整改效果
- [ ] 确认问题关闭
- [ ] 出具审计报告

### Step 5: 合规认证

- [ ] 获取合规认证
- [ ] 归档合规材料
- [ ] 建立合规档案

## 常见合规检查项

| 检查项 | 检查内容 |
|--------|---------|
| 访问控制 | 权限管理、账号安全 |
| 数据保护 | 数据加密、备份恢复 |
| 日志审计 | 操作日志、安全日志 |
| 变更管理 | 变更审批、变更记录 |
| 应急预案 | 故障预案、安全预案 |

## 输出物

| 输出物 | 格式 | 存放位置 |
|--------|------|---------|
| 合规检查清单 | 文档 | 项目文档库 |
| 审计报告 | 文档 | 项目文档库 |
| 整改记录 | 文档 | 项目文档库 |
