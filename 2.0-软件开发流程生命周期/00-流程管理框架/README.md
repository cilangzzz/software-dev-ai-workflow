# 流程管理框架

## 概述

流程管理框架定义了AI驱动软件开发工作流的流程治理机制，确保流程的标准化、可执行性和持续改进。

## 核心文档

| 文档 | 说明 |
|------|------|
| [流程治理规范](流程治理规范.md) | 流程的定义、审批、发布、变更、废弃的生命周期管理规范 |
| [流程角色职责](流程角色职责.md) | 流程负责人、流程执行者、流程审计者等角色的职责定义 |
